# 商品发布与询盘转化

> 分类：05-商品发布  
> 本页更新：2026-05-26

## 核心认知

B2B 商品发布的目标不是把页面写得花，而是让买家快速判断：这个供应商能不能做我要的产品、能不能按我的规格交付、是否值得发询盘。

## 商品页结构

| 模块 | 作用 |
|---|---|
| 标题 | 覆盖核心关键词、材质、用途和定制能力 |
| 主图 | 让买家一眼识别产品和应用场景 |
| 参数 | 规格、材质、尺寸、颜色、认证、包装 |
| MOQ | 明确最小起订量和样品政策 |
| 价格 | 给区间或阶梯价，避免误导 |
| 详情 | 展示工厂能力、生产流程、质检和案例 |
| 交付 | 交期、包装、物流方式和付款条件 |

## 标题公式

```text
核心品类 + 关键材质/功能 + 使用场景 + 定制/OEM能力
```

示例：

```text
Custom Stainless Steel Water Bottle for Outdoor Sports OEM Logo
```

## 询盘转化

询盘回复要解决三个问题：

1. 买家要什么？
2. 我们能不能做？
3. 下一步怎么推进？

高质量回复通常包含：

- 对买家需求的复述。
- 关键规格确认。
- 价格区间或报价条件。
- MOQ、样品、交期和包装。
- 需要买家补充的信息。
- 下一步动作，如发目录、确认图纸、安排样品。

## 跟进节奏

- 0 到 2 小时：首次响应，确认需求。
- 24 小时内：补充报价、目录、案例或样品方案。
- 3 天内：二次跟进，询问规格和决策周期。
- 7 天内：提供替代方案、热销款或低 MOQ 方案。
- 长期：按客户行业和采购周期进入 CRM。
