# 询盘分层与报价 SOP

> 分类：06-询盘转化  
> 本页更新：2026-05-26

## 询盘分层

| 等级 | 特征 | 处理方式 |
|---|---|---|
| A 类 | 需求清晰、数量明确、公司信息完整、有采购时间 | 优先报价，业务员重点跟进 |
| B 类 | 有真实需求，但规格或预算不完整 | 引导补充信息，提供区间报价 |
| C 类 | 只问价格、无数量、无公司信息 | 标准回复，低成本跟进 |
| D 类 | 明显套价、无关产品、异常地区或风险高 | 谨慎处理或归档 |

## 报价前确认

- 产品规格、材质、尺寸、颜色。
- 数量、MOQ、是否长期采购。
- 包装、贴标、定制要求。
- 目标市场和认证要求。
- 交期、贸易条款、物流方式。
- 付款方式和样品需求。

## 报价结构

```text
有效期：
备注：
```

## 跟进原则

- 不只发价格，要推动买家补齐规格。
- 报价要写清前提，避免后续争议。
- 高质量客户进入 CRM，记录每次沟通。
- 多轮询盘要形成版本管理，避免报价混乱。
