# 数据分析基础

> 分类：10-数据分析  
> 本页更新：2026-05-26

## 指标框架

| 阶段 | 指标 |
|---|---|
| 流量 | 曝光、点击、访客、关键词排名 |
| 询盘 | 询盘数、有效询盘率、RFQ 数 |
| 响应 | 回复时效、报价完整度、跟进次数 |
| 转化 | 样品转化、订单转化、成交额 |
| 履约 | 准时发货率、质检通过率、争议率 |
| 客户 | 复购率、客户国家、老客户占比 |

## 分析思路

- 曝光低：检查关键词、类目、产品覆盖和广告。
- 点击低：检查主图、标题和价格吸引力。
- 询盘低：检查详情页、MOQ、信任背书和产品匹配。
- 转化低：检查报价、响应、样品和客户质量。
- 复购低：检查履约、质量和客户维护。
