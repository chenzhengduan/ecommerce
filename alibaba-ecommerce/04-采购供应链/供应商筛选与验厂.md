# 供应商筛选与验厂

> 分类：04-采购供应链  
> 本页更新：2026-05-26

## 核心认知

供应商筛选不是找最低价，而是判断对方是否能长期稳定交付。阿里体系里的供应商可能是工厂、贸易商、档口、品牌代理或服务商，角色不同，能力边界也不同。

## 筛选维度

| 维度 | 检查点 |
|---|---|
| 主营能力 | 是否长期做目标品类，产品线是否聚焦 |
| 生产能力 | 设备、产线、产能、工人、旺季交付 |
| 质量能力 | 来料检验、过程检验、出货检验、不良处理 |
| 合规能力 | 认证、检测报告、出口资质、知识产权意识 |
| 沟通能力 | 响应速度、专业度、报价完整度 |
| 配合能力 | 打样、定制、包装、贴标、售后 |
| 风险水平 | 交期波动、低价异常、资质不清、频繁换主体 |

## 验厂层级

- 远程初筛：营业执照、店铺信息、资质、产品图、视频、历史交易。
- 视频验厂：办公室、仓库、产线、设备、样品间、质检区域。
- 第三方验厂：适合高金额、长期合作、定制品和高风险品类。
- 驻厂或代理跟单：适合复杂供应链和连续订单。

## 输出物

每个重点供应商建议建立一页档案：

- 公司名称、联系人、地址、主营品类。
- 可做产品、MOQ、报价、交期。
- 样品记录、质检记录、问题记录。
- 合作等级：试用、稳定、战略、淘汰。
