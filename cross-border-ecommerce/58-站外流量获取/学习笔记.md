# 58-站外流量获取

## 主题概述

站外流量（Off-site Traffic）指通过第三方渠道引导用户访问店铺或产品的流量，包括社交媒体广告、搜索引擎、内容营销、网红营销、Deal论坛、私域流量和邮件营销等。

## 核心要点

### 主要渠道
1. **社交媒体广告**：Facebook/Instagram、TikTok、Pinterest、YouTube、Snapchat
2. **搜索引擎**：Google SEO、Google Ads、Bing
3. **内容营销**：博客、测评网站、Deal论坛
4. **网红/KOL**：社交达人、YouTuber、博主合作
5. **私域流量**：WhatsApp群组、Facebook群组
6. **邮件营销**：EDM自动化（Mailchimp/Klaviyo）

### 关键指标
- CPM、CPC、CPA、ROAS
- UTM追踪链接（Bitly/Tapfiliate）

### 站外与站内关系
- 站外流量可提升产品在平台内的排名
- 独立站+平台双渠道分流降低成本
- 需合规操作，避免被视为操纵排名

### 实操步骤
确定目标 → 选渠道 → 准备素材 → 设置追踪 → 小预算测试 → 数据复盘
