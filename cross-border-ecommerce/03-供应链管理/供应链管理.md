# 供应链管理

> 分类：03-供应链管理 | 学习日期：2026-04-20

## 供应链核心链路

```
工厂/供应商 → 头程物流（国内）→ 中转仓/第三方仓库 → 出海航运/空运 → 目的国仓库（海外仓/FBA）
```

## 供应商开发渠道

| 渠道 | 特点 | 适合阶段 |
|------|------|----------|
| **1688** | 品类全，价格透明，但质量参差 | 测款、起步 |
| **阿里巴巴国际站** | 有外贸经验的工厂，适合定制 | 有一定订单量 |
| **工厂拜访（广交会）** | 直接看厂，深度合作 | 稳定大货阶段 |
| **Google搜索** | 找品牌代工厂/OEM | 差异化产品 |
| **跨境供应链公司** | 提供选品+物流+资金一条龙 | 新手过渡 |

## 供应商筛选标准

- **MOQ（最小起订量）**：小批量测试阶段，选MOQ低的供应商（50-100件为佳）
- **打样周期**：3-7天为正常，超过14天需要谨慎
- **样品质量**：必须亲自检验，不要只看图片和视频
- **生产能力**：月产能是否匹配你的备货需求
- **响应速度**：24小时内回复是基本，48小时以上需要谨慎
- **退换货政策**：大货出问题能否退换，是合作前提

## 产品成本拆解

**单品成本 = 出厂价 + 包装费 + 质检费 + 装箱费**

| 成本项 | 说明 |
|--------|------|
| **出厂价** | 基础产品成本 |
| **包装（内盒/外箱）** | 亚马逊要求包装精美，建议单独计算 |
| **质检费** | 批量出货前的检验费用 |
| **物流到仓库** | 从工厂运到头程仓库的国内物流 |

## 头程物流成本计算

**头程 = 国内仓库操作费 + 国际运费 + 清关费 + 目的港费用**

- **空运**：7-10天到达，适合高价值/紧急补货，费用约30-60元/kg
- **海运（散货）**：25-35天到达，适合大批量备货，费用约5-15元/kg
- **海运（整柜）**：量大时成本最低，20GP/40GP/40HC柜型
- **铁路**（中欧班列）：18-25天，约10-25元/kg，比空运便宜比海运快

## 库存管理

- **安全库存计算**：安全库存 =（日均销量 × 备货周期） × 1.2（保险系数）
- **断货影响**：Listing排名下滑，广告权重降低，亚马逊系统降权
- **滞销处理**：长期仓储费（亚马逊按立方英尺收），及时清理或移仓
- **FBA库存警示**：提前30天关注，大促前备货要更提前

## 质检流程

1. **首批全检**：第一批货全部检查，确认供应商质量稳定
2. **中期抽检**：每批次抽检10-20%，主要查外观和功能
3. **第三方质检**（SGS/BV）：大批量时使用，费用高但有公信力
4. **验货标准文件**（IQC/QC Checklist）：发货前让供应商签字确认
