## 一、学习内容摘要

### 1. 核心广告效果指标

**ACoS（广告销售成本）**
- 定义：广告支出 ÷ 广告销售额 × 100%
- 公式：ACoS = (广告支出 / 广告销售额) × 100%
- 用途：衡量广告盈利能力，ACoS越低说明广告效率越高
- 盈亏平衡点：ACoS = 产品毛利率时为盈亏平衡

**ROAS（广告支出回报率）**
- 定义：广告收入 ÷ 广告支出
- 公式：ROAS = 广告销售额 / 广告支出
- 用途：衡量每花费1美元能带来多少收入，ROAS>1才能盈利

**CTR（点击率）**
- 定义：点击次数 ÷ 曝光次数 × 100%
- 优质水平：一般>0.5%，优秀>1%

**CVR（转化率）**
- 定义：订单数 ÷ 点击次数 × 100%
- 行业平均水平：约3-5%

**PPC（每次点击付费）**
- 亚马逊站内广告的核心计费方式

### 2. 广告报表类型

**广告活动报表（Campaign Report）**
- 最高层级，了解整体广告活动表现
- 包含：曝光量、点击次数、CTR、花费、ACoS、ROAS

**广告组报表（Ad Group Report）**
- 中间层级，分析同一广告组内不同产品的表现

**关键词报表（Search Term Report）**
- 最细粒度，了解哪些搜索词带来了转化
- 用于否定关键词优化和关键词拓展

**商品报表（Product Report）**
- 分析广告展示的商品表现
- 哪些ASIN获得了更多曝光和点击

### 3. 数据分析维度

**时间维度分析**
- 按日/周/月分析广告表现趋势
- 识别季节性波动和异常数据

**横向对比**
- 不同广告活动之间对比
- 不同产品之间对比
- 不同时期对比

**归因分析**
- 直接转化 vs 间接转化
- 广告对自然搜索的影响

## 二、知识验证

| 判断项 | 结果 | 说明 |
|--------|------|------|
| ACoS公式正确 | ✓ | 广告支出/广告销售额×100% |
| ROAS公式正确 | ✓ | 广告销售额/广告支出 |
| ACoS越低越好 | ✓ | 前提是保持正向ROAS |
| CTR和CVR关系 | ✓ | CTR影响流量，CVR影响转化 |

## 四、实战应用建议

1. **每日监控**：曝光量、点击次数、花费、ACoS
2. **每周分析**：关键词表现、搜索词报表
3. **每月复盘**：整体ROAS、广告投入产出比
4. **优化方向**：
   - ACoS高于毛利率 → 优化出价或关键词
   - CTR低 → 优化主图和标题
   - CVR低 → 优化Listing或价格
