# 235 - 亚马逊选品之广告AI智能化投放与预算优化系统

## 学习内容摘要

### 核心概念

1. **AI广告投放的定义**
   - 利用人工智能和机器学习算法自动优化亚马逊广告投放
   - 包括自动竞价、智能定向、预算分配、文案生成等

2. **主要AI广告工具类型**
   - 自动竞价（Automatic Bidding）：根据转化率自动调整出价
   - 智能定向（Smart Targeting）：AI识别高转化人群
   - 预算优化（Budget Optimization）：自动分配预算到高效广告位
   - 文案生成AI：自动生成广告文案和标题

3. **预算优化策略**
   - 设置日预算和生命周期预算
   - 利用AI预测最佳投放时间
   - 动态调整预算应对竞争变化
   - 分Campaign管理不同产品线

4. **ACoS优化目标**
   - 目标ACoS（Target ACoS）设置
   - 平衡ACoS与ROI
   - 新品推广期允许较高ACoS
