# 亚马逊选品之专利排查与侵权规避

## 主题概述

专利排查与侵权规避是亚马逊选品中最关键的合规环节之一。忽视专利问题可能导致产品下架、店铺被封、资金冻结，甚至面临法律诉讼。

## 核心知识点

### 1. 专利类型

| 类型 | 保护内容 | 保护期限 |
|------|----------|----------|
| 发明专利 | 产品技术方案、功能创新 | 20年 |
| 外观设计专利 | 产品外观造型、图案 | 15年（美国15年/欧盟25年） |
| 实用专利 | 产品的结构、构造创新 | 10-15年 |

### 2. 专利排查渠道

- **USPTO**（美国专利商标局）：https://patft.uspto.gov/
- **EPO**（欧洲专利局）：https://worldwide.espacenet.com/
- **WIPO**：https://www.wipo.int/patentscope/
- **Google Patents**：https://patents.google.com/
- **中国专利查询**：https://cpquery.cnipa.gov.cn/

### 3. 排查关键步骤

1. **确定产品核心关键词** → 提取技术特征
2. **多国检索** → 目标销售市场专利都要查
3. **检索同品类竞品** → 查找相似专利
4. **关注专利有效期** → 确认是否在保护期内
5. **聘请专业机构** → 复杂产品建议外包

### 4. 侵权风险类型

- **直接侵权**：销售含他人专利的产品
- **间接侵权**：诱导、帮助他人侵权
- **外观侵权**：相似度>60%可能构成侵权

### 5. 规避策略

| 策略 | 说明 |
|------|------|
| 专利回避设计 | 改变产品形状、结构、功能实现方式 |
| 寻找公版/公模 | 选择无专利保护的产品公版 |
| 自主研发 | 申请自己的专利形成保护壁垒 |
| 专利许可 | 获得专利权人授权（成本高） |
| 购买专利 | 购买相关专利所有权 |

### 6. 侵权应对措施

- 收到TRO（临时限制令）立即下架产品
- 冻结资金勿擅自转移
- 寻求专业律师帮助
- 评估是否和解或应诉
