# 121-亚马逊选品之产品合规与认证要求

## 学习内容摘要

### 1. 常见认证类型
- **CPC认证**（儿童产品证书）：美国亚马逊要求儿童产品必须提供
- **CE认证**：欧洲市场准入门槛
- **FCC认证**：电子产品电磁兼容性要求
- **FDA认证**：食品、药品、化妆品等
- **UL认证**：电子电气产品安全认证

### 2. 目标市场合规要求
- **美国**：CPSC（消费品安全委员会）监管严格
- **欧洲**：CE标志、RoHS指令、WEEE指令
- **日本**：PSE标志、食品接触材料法规
- **各站点需分别认证，不能通用**

### 3. 产品标签要求
- 原产国标识
- 年龄警告标签
- 材质成分标签
- 制造商信息

### 4. 禁限售商品
- 亚马逊各站点禁限售商品列表不同
- 需定期查看卖家平台公告

### 5. 知识产权风险
- 商标侵权排查
- 专利侵权排查
- 品牌备案重要性

---

## 对不对的判断：✓ 正确

以上内容是基于亚马逊平台常见合规要求的总结。不同品类、不同市场的具体要求确实各有不同，需要针对性研究。

---

## 是不是新知识：是

对于跨境电商从业者来说，产品合规与认证是一个必须掌握但可能之前了解不深的领域。之前学习选品时更多关注市场数据，对合规性重视不够。

---

## 是否好理解：较好理解

概念清晰，逻辑顺畅。认证类别和市场要求都比较具体，容易掌握。唯一需要注意的是各国认证标准会更新变化，需要持续关注。
