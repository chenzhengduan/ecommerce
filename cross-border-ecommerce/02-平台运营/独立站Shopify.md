# 独立站建站与推广（Shopify）

> 分类：02-平台运营 | 学习日期：2026-04-20

## 独立站 vs 平台

| 对比维度 | 独立站 | 平台（Amazon/速卖通）|
|----------|--------|----------------------|
| 流量 | 自身引流（SEO/广告/社媒）| 平台自带流量 |
| 数据所有权 | 完全掌控用户数据 | 不归卖家所有 |
| 规则约束 | 完全自主 | 受平台规则限制 |
| 信任度 | 需要自己建立 | 买家信任平台 |
| 起步难度 | 高（需要引流）| 低（有现成流量）|
| 费用结构 | 平台费+广告费 | 佣金+广告费 |

## Shopify 建站核心步骤

1. **注册账号** → 选择月租套餐（$29/$79/$299）
2. **选择主题** → 免费主题（Dawn/Boundless）或付费主题
3. **安装必备App** → 支付（Stripe/PayPal）、物流（AfterShip）、邮件（Klaviyo）
4. **绑定域名** → 购买域名并解析（Route53/Namecheap）
5. **配置支付** → Shopify Payments（支持信用卡）或Stripe/PayPal
6. **设置运费** → 按地区/重量设置运费规则
7. **完善政策** → 退款政策、隐私政策、服务条款（法律要求）
8. **测试下单** → 用测试信用卡跑通全流程

## 独立站流量体系

```
流量 = 付费流量（广告）+ 免费流量（SEO/社媒）+ 私域流量（邮件/社媒）
```

## Google 广告

| 广告类型 | 特点 | 适合阶段 |
|----------|------|----------|
| **搜索广告（Search）** | 关键词精准投放 | 主动需求流量 |
| **展示广告（Display）** | 图片banner广告 | 品牌曝光 |
| **购物广告（Shopping）** | 产品图文广告 | 电商转化 |
| **效果最大化广告（PMax）** | AI自动优化全渠道 | 中后期跑量 |

- **Google Analytics 4（GA4）**：必须安装，跟踪用户行为和转化路径
- **GMC（Google Merchant Center）**：上传产品 Feed，购物广告必填

## Facebook/Instagram 广告

- **像素（Pixel）**：安装到网站，跟踪转化事件（加购/结账/购买）
- **受众定向**：
  - 类似受众（Lookalike）：基于现有客户找相似人群
  - 兴趣定向：按兴趣/行为定位
  - 再营销（Retargeting）：访问过网站但未购买的用户
- **广告素材**：视频>图片>轮播，TikTok风格创意在Instagram Reels很有效
- **ROAS目标**：跨境电商通常要求ROAS>3（即花$1广告带来$3销售额）

## TikTok 电商

- **TikTok Shop**：直播电商+短视频挂车，适合冲动消费品
- **TikTok Ads**：信息流广告，品牌曝光
- **红人营销**：找中腰部KOC合作（10万-100万粉），成本低转化好
- **内容关键**：本土化内容，不要有中国元素（翻译腔、中文标识）

## SEO 基础

- **On-Page SEO**：Title、H1、Meta Description、图片ALT标签、URL结构
- **技术SEO**：网站速度（<3秒）、移动端适配、HTTPS安全认证、站点地图
- **内容SEO**：写博客文章（Buyers Guide/How-to），获取自然搜索流量
- **外链建设**：获取高质量外链（Guest Post、行业目录）
