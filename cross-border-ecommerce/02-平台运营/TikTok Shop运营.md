# TikTok Shop 运营

> 学习日期：2026-04-20

## 核心要点

- TikTok Shop 是 TikTok 内置电商功能，支持短视频/直播带货
- 两种模式：内容电商（冲动消费）和货架电商（搜索购买）
- 全托管模式：商家供货，TikTok 负责运营，适合工厂型商家
- 自运营模式：商家自主运营，适合有品牌运营能力的商家
- 东南亚是最佳入门市场，美国市场增长快但有政策风险
- 选品核心：视觉化、新奇特、$5-20 价格带
- 内容必须本地化，避免明显的"中国式英语"
- 达人合作是核心流量来源，纯佣起步风险最低
- 政策风险：美国封禁风险持续关注
