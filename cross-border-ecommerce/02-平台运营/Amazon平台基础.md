# Amazon 平台基础

> 分类：02-平台运营 | 学习日期：2026-04-20

## Amazon 卖家账号体系

### 两类账号

| 类型 | 全称 | 说明 |
|------|------|------|
| **VC** | Vendor Central | 邀请制，亚马逊直接采购，品牌方 |
| **SC** | Seller Central | 第三方卖家，大多数人的选择 |

### 两种销售方式

| 方式 | 说明 | 适合谁 |
|------|------|--------|
| **FBA** | Fulfillment by Amazon，备货到亚马逊仓库，由亚马逊负责发货、退货、客服 | 新手、怕麻烦的卖家 |
| **FBM** | Fulfilled by Merchant，自己发货，亚马逊只负责展示 | 有自建物流能力、卖大件商品的卖家 |

## FBA 完整流程

1. **注册卖家账号** → 准备营业执照、双币信用卡、收款账户
2. **选品** → 核心是调研竞品销量、评价、价格
3. **创建 listing** → 标题、五点描述、图片、A+ 内容
4. **发货到亚马逊仓库** → 创建 FBA 发货计划，打标签，装箱
5. **亚马逊上架销售** → 入库后开始销售
6. **运营优化** → 广告、评价、排名

## Amazon 核心指标（卖家必须懂）

| 指标 | 含义 | 重要原因 |
|------|------|----------|
| **BSR** | Bestseller Rank，类目畅销榜排名 | 数字越小卖得越好 |
| **FBA fees** | 亚马逊收取的履约费 | 直接影响利润 |
| **PPC** | Pay Per Click，站内广告 | 主要流量来源 |
| **ACOS** | Advertising Cost of Sales，广告销售比 | 越低说明广告效率越高 |
| **Buy Box** | 黄金购买按钮，占领它才有大部分销量 | 核心竞争点 |

## 新手常见误区

1. **盲目选品** — 不做调研就上大量库存
2. **忽视类目审核** — 某些类目需要亚马逊审批（母婴、玩具等）
3. **图片太差** — 主图白底是基本要求，不是加分项
4. **不懂 ACOS 就开广告** — 广告烧钱但不带来利润
5. **不关注竞争对手** — 亚马逊是竞争激烈的市场

## 开店基本要求（中国卖家）

| 要求 | 细节 |
|------|------|
| 营业执照 | 必须是真实有效的（亚马逊会验证） |
| 双币信用卡 | 用于扣月租（39.99美元/月） |
| 收款账户 | 派安盈、万里汇、pingpong 等 |
| 税号 | 美国需要 EIN 或 ITIN |
| 干净的电脑和网络 | 避免账号关联问题 |

## Amazon vs 其他平台（Shopify/速卖通）

| 对比 | Amazon | Shopify | 速卖通 |
|------|---------|---------|--------|
| 流量 | 平台自带巨大流量 | 自身引流 | 平台流量 |
| 自主性 | 低（规则多） | 高（完全自己掌控） | 低 |
| 信任度 | 极高（买家信任亚马逊） | 取决于卖家 | 中等 |
| 收费 | 月租 + FBA费用 + 佣金 | 月租 + 交易费 | 佣金较低 |
| 适合 | 有好产品想快速出单 | 想建立品牌 | 卖低价商品 |
