# 123-亚马逊选品之爆款复制与横向扩展策略

## 学习内容摘要

### 什么是爆款复制策略？

爆款复制（Copycat/Scale Strategy）是指将在一个类目或产品线上成功验证的爆款，通过产品变体、功能迭代或品类延伸的方式，复制到更多SKU或市场的运营策略。

### 核心方法

1. **变体扩展（Variations）**
   - 颜色变体：从1个颜色扩展到5-10个颜色
   - 尺寸变体：S/M/L/XL 或 单一尺寸测试
   - 套装组合：单个装 → 2件装 → 4件装 → 套装
   
2. **产品线延伸**
   - 向上延伸：低端 → 中端 → 高端
   - 向下延伸：高端 → 性价比版本
   - 横向扩展：从核心产品延伸相关配件

3. **市场复制**
   - 站点复制：美国站爆款→欧洲站/日本站
   - 平台复制：亚马逊爆款→独立站/eBay

### 关键步骤

1. **验证爆款**：确保原产品BSR排名前50，月销500+才能复制
2. **数据驱动**：通过Helium 10/Jungle Scout验证市场需求
3. **小步快跑**：先测试2-3个变体，再大规模扩展
4. **供应链保障**：确保产能和交期跟上

### 风险控制

- 不要一次性上线所有变体，分批测试
- 监控每个变体的转化率，及时淘汰低表现SKU
- 预留广告预算用于新品冷启动
