# 主题204：亚马逊仓储与物流成本优化深度攻略

> 学习时间：2026-04-22 11:16

## 学习内容摘要

### 一、FBA仓储费结构详解（2026年最新）

**月度仓储费（Standard Size）:**
- 1月-9月：$0.78/cu.ft（每月调整）
- 10月-12月旺季：$2.40/cu.ft（约为平时的3倍）

**长期仓储费（Long-Term Storage）:**
- 存放181-365天：$6.90/cu.ft 或 $0.15/unit（取较高者）
- 存放365天以上：$15.00/cu.ft 或 $0.70/unit（取较高者）

**移除/销毁费：**
- 移除费：$0.50-$0.60/标准件
- 销毁费：约$0.30/标准件（部分地区略低）

**超容费（FBA Overage）：** 2024年起对超容量的额外费用。

### 二、仓储成本优化核心策略

**1. 库龄管理（核心）**
- 库龄越短，仓储费越低，残值回收越快
- 超过90天的滞销库存立即启动清仓
- 定期查看FBA Inventory Dashboard的DoS（Days of Supply）指标
- 目标：70%以上库存在30天内售出

**2. 库容优化**
- 合理使用仓储利用箱（Stocker），避免空间浪费
- 标准件vs大件混用导致空间浪费
- 使用FBA Box Level Configuration优化箱内空间
- 避免"空气包装"——用小箱装轻小商品

**3. 入仓时间策略**
- 避开10月-12月旺季前大量入库（旺季仓储费是平时的3倍）
- Q4旺季前尽量清完旧货，减少旺季仓储费压力
- 利用淡季（1月-6月）多备货，旺季（7月-9月）控制库存水位

**4. 库存健康分级**
- A类（0-30天）：正常运营，优质库存
- B类（31-90天）：监控，避免成为C类
- C类（91-180天）：清仓预警，启动移除或促销
- D类（181-365天）：立即清仓，高额长期仓储费
- F类（365天+）：超高费用，必须立即处理

### 三、滞销库存处理方案

**1. FBA Liquidations（清仓）**
- 通过亚马逊官方清仓项目，将滞销库存批量出售给清仓商
- 回收率通常为原价5%-20%
- 优点：简单快捷，无需自己处理库存
- 缺点：回收价低，可能影响利润
- 适合：库龄181天以上的滞销品

**2. 促销清仓（Remove Order）**
- 创建奥特莱斯（Outlet）促销：专为清仓设计的亚马逊促销页面
- 社交媒体促销代码：在Facebook/Instagram发布折扣码
- Deal平台（Slickdeals/Dealabs）发帖清仓
- 站外清仓+站内评价同步提升

**3. 主动移除（Remove）**
- 当清仓价值低于移除费用时，直接移除更划算
- 移除后可发往海外仓重新包装再入仓
- 或发送给第三方清仓商（如B-Stock/Liquidation.com）

**4. 库龄自动移除（LTSF - Long-Term Storage Fee）**
- 设置自动移除政策，避免高额长期仓储费
- 超过365天库存自动移除或清仓
- 建议对D类和F类库存启用自动移除

### 四、物流方案成本对比

| 方案 | 适用场景 | 仓储成本 | 物流成本 | 适合品类 |
|------|---------|---------|---------|---------|
| FBA | 追求Prime标志，标准化管理 | 高（旺季3倍） | 中等 | 标品，重量<2kg |
| FBM | 不想压货，大件商品 | 低/无 | 较高 | 大件，定制商品 |
| 第三方海外仓 | 批量备货，多平台发货 | 中等 | 中等 | 多平台卖家 |
| 专线小包 | 低价值商品，样品 | 无 | 低 | 低价值轻小商品 |

### 五、库存周转优化模型

**核心公式：**
- 安全库存 = 日均销量 × 备货天数 + 安全系数
- 备货天数 = 生产周期 + 运输周期 + 入仓处理时间
- 避免断货（影响排名）同时避免积压（仓储费+残值风险）

**ABC分类管理：**
- A类（爆款，20% SKU贡献80%销量）：高安全库存，优先补货
- B类（稳定款）：中等库存，定期监控
- C类（长尾款）：低库存，按订单备货

### 六、实战避坑指南

1. **旺季前清仓**：Q4前（9月底前）必须将180天以上库存清完
2. **避免"空气箱"**：轻小商品用小箱，避免按标准件收费却只用一半空间
3. **不要低估移除费**：长期仓储费往往比移除费+清仓价值更高
4. **定期巡检DoS**：每周检查DoS>90天的SKU，及时处理
5. **FBA费用变更通知**：每年旺季前亚马逊可能调整仓储费，需提前关注
