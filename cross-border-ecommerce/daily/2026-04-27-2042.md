## 学习内容摘要

### 1. 库存预测的核心概念
- **库存预测**：基于历史销售数据、市场趋势、季节性因素，预测未来需求量
- **目的**：避免缺货（丢失销售）或过量库存（增加仓储费）

### 2. 预测方法

#### 基础方法
- **移动平均法**：计算过去N天的平均销量
- **季节性调整**：考虑节假日、促销季的影响
- **趋势分析**：判断产品是上升期、平稳期还是衰退期

#### 高级方法
- **机器学习预测**：利用AI模型分析更多变量
- **多维度分析**：竞品排名、评论数量变化、广告投放变化

### 3. 关键指标

| 指标 | 说明 |
|------|------|
| 日均销量（ADS） | 过去7/14/30天的平均销量 |
| 库存周转率 | 销量/平均库存 |
| 补货周期 | 从下单到入库的时间 |
| 安全库存 | 预防缺货的缓冲库存 |

### 4. 补货公式

```
补货量 = (预计日均销量 × 补货周期) + 安全库存 - 当前库存 - 在途库存
```

### 5. 工具推荐
- **亚马逊库存绩效指标（IPI）**：监控库存健康度
- **第三方工具**：Jungle Scout、Helium 10、Inventory Lab
- **亚马逊API**：可接入自有系统
