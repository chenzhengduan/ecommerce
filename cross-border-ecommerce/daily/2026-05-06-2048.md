## 学习内容摘要

### 1. 定价策略核心要素
- **成本加成定价**：基础成本 + 预留利润margin + 平台费用 + 物流成本
- **市场导向定价**：参考竞争对手价格，根据差异化调整
- **价值导向定价**：根据客户感知价值定价，高于成本

### 2. 跨境电商定价需考虑因素
- 产品成本（采购/生产成本）
- 国际物流费用（头程+尾程）
- 平台佣金（亚马逊约15%，独立站无佣金但需支付手续费）
- 营销推广费用（广告预算）
- 退货率预留（服装10-20%，电子3-5%）
- 各国 VAT/关税（中国出口一般免关税，美国$800内免税）
- 汇率波动风险准备金

### 3. 定价公式
```
售价 = (成本 + 运费 + 平台费 + 利润) / (1 - 预期退货率)
```

### 4. 价格测试与优化
- A/B测试不同价格点
- 分析不同价格区间的转化率
- 寻找最优价格点（利润最大化）
- 区分引流款（低利润/无利润）和利润款（正常利润）
