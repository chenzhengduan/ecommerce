# 150-亚马逊店铺数据分析与报表可视化

## 学习内容摘要

亚马逊店铺数据分析与报表可视化是运营Seller Central的核心能力，主要包括：

### 1. 核心数据看板
- **业务数据**：销售额、订单量、转化率、退货率、营收
- **广告数据**：ACoS、RoAS、CPC、曝光量、点击量、转化量
- **库存数据**：FBA库存、在途数量、可售天数、IPI分数
- **财务数据**：收入、支出、利润、平台费用

### 2. 数据报表工具
- **Amazon Seller Central原生**：业务报告、广告报告、库存报告
- **第三方工具**：Helium 10、Jungle Scout、Keepa、数据牛
- **自定义报表**：通过SP-API或数据下载定制

### 3. 可视化关键指标
- **Dashboard设计原则**：KPI核心、趋势对比、多维度下钻
- **常用图表**：折线图（趋势）、柱状图（对比）、饼图（占比）、热力图（时机）
- **数据更新频率**：实时/日更/周更/月报

### 4. 数据驱动决策流程
1. 数据采集 → 2. 清洗处理 → 3. 分析解读 → 4. 可视化呈现 → 5. 行动建议

## 对不对的判断

✓ 基本正确。Seller Central确实提供完整的数据报表功能，第三方工具可以增强可视化能力。核心指标和流程符合业界实践。
