# Topic #141: 亚马逊仓储费节省策略与FBA成本优化实战（2026年费用变更/仓储利用箱/库存构型优化/FBA价值修复）

## 学习主题
亚马逊仓储费节省策略与FBA成本优化实战（2026年费用变更/仓储利用箱/库存构型优化/FBA价值修复）

## 学习内容摘要

### 一、2026年FBA费用变更概览
亚马逊每年调整FBA费用，2026年主要变化：
- **仓储费上调**：标准件仓储费持续上涨，旺季（10-12月）仓储费是淡季的2-3倍
- **库存超量费（Overage Fees）**：对超过容量限制的库存收取高额超量费
- **FBA移除/处置费调整**：移除费用上涨，处置费也有调整
- **入库配置服务费**：2024年引入，按库存配置方式收费（分仓vs.合仓）

### 二、仓储利用率优化（立方英寸管理）
**核心逻辑**：FBA按立方英尺收费，优化仓储利用率直接降低成本。

1. **产品尺寸分级**：
   - A尺寸（标准件）：≤15" x 6" x 0.75"
   - B尺寸（大标准件）：≤18" x 14" x 0.8"
   - C尺寸（大件）：≤59" x 30" x 0.75"
   
2. **仓储利用箱（Storage Utilization Bucket）**：
   - 亚马逊根据每个SKU的立方英寸平均值计算"平均仓储利用"
   - 低于平均利用率的SKU会被收取更高单位费用
   - 优化方法：将小件产品与大件产品组合成Bundle，提升单件体积利用率

3. **单位体积收益优化**：
   - 优先发小而高价值的产品
   - 避免发送"又大又便宜"的产品（仓储费吃利润）

### 三、库存构型优化（Inventory Placement）
**入库配置服务费（Inventory Placement Service Fee）**：
- 选择"分布式混合（Distributed Inventory）"：发往多个运营中心，费用较低
- 选择"库存配置（Inventory Placement）"：发往单个运营中心，每件额外收费约$0.30-$0.60

**决策框架**：
- SKU多、量小 → 选分布式混合，降低配置费
- SKU少、量大 → 选库存配置，节省移仓成本

### 四、FBA价值修复工具
1. **FBA Liquidations（批量清仓）**：
   - 亚马逊以原价的5%-20%收购滞销库存
   - 比处置费（$0.15-$0.30/件）更划算
   - 适合：冗余库存、季节性产品过季
   
2. **FBA Donations（捐赠）**：
   - 部分产品可捐赠，避免销毁同时获得税收优惠
   
3. **FBA Grade and Resolve（评级转售）**：
   - 对退回的顾客产品进行评级，以较低价格转售

4. **移除 vs. Liquidations vs. 处置**：
   - 移除费：发回国内，费用最高
   - Liquidations：亚马逊收购，中等收益
   - 处置费：最低，但零收益

### 五、IPI评分优化与仓储容量管理
- **IPI（Inventory Performance Index）**目标：≥500
- 影响IPI的因素：
  - 冗余库存比率（Surplus Inventory）
  - 无在售库存比率（Unsellable Inventory）
  - 在售库存比率（In-stock Rate）
  - FBA库存率（FBA Inventory Age）
  
- **Capacity Manager**：旺季可申请额外容量，需提前规划

### 六、日常成本优化实操
1. **最小起订量+空运/海运组合**：低价值产品发海运，高补货速度产品发空运
2. **淡季提前规划**：旺季前（9月前）清理冗余库存，避免旺季仓储费
3. **多地点配送（MFN + FBA混合）**：对低周转产品用自发货，降低FBA仓储费
4. **产品包装优化**：减少包装体积（符合亚马逊要求的前提下）
5. **季节性/节日产品提前清仓**：节后立即申请Liquidations或移除
