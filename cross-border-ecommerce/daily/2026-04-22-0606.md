## 学习内容摘要

### 一、亚马逊库存绩效指标体系（IPI Score）

**IPI（Inventory Performance Index）评分机制：**
- 亚马逊于2024年起将IPI门槛从400提高到500（美国站）
- 低于500分的账号会受到库存容量限制
- IPI分数每月更新一次（非实时）

**IPI四维评分体系：**
1. **FBA库存率**（In Stock Rate）：有现货产品占总SKU的比例，目标>90%
2. **滞销库存率**（Stranded Inventory Rate）：无在售信息的库存，目标<5%
3. **库存周转**（Sell-through Rate）：月销量/月均库存，目标因类目而异
4. **FBA仓库利用率**（Utilization Rate）：使用立方英尺/限制容量，目标<50%最佳

**提升IPI策略：**
- 及时修复Stranded Inventory（设置正确Listing信息）
- 清理超龄库存（超过365天的库存影响严重）
- 保持合理库存深度（避免断货但也不积压）
- 使用亚马逊库存规划工具预测需求

### 二、亚马逊库存管理核心指标

**DoS（Days of Supply）可售天数：**
- 计算公式：当前库存 ÷ 日均销量
- 理想区间：28-60天
- <7天：严重断货风险
- >90天：库存积压

**库龄分级（Inventory Age）：**
| 库龄 | 风险等级 | 影响 |
|------|---------|------|
| 0-90天 | 低风险 | 正常 |
| 91-180天 | 中风险 | 开始产生长期仓储费 |
| 181-270天 | 高风险 | 长期仓储费翻倍 |
| 271-365天 | 极高风险 | 库存将产生高额费用 |
| >365天 | 超龄库存 | 可能被强制清理 |

**库存健康状态类型：**
- **Healthy**（健康）：DoS在28-60天
- **Excess**（过剩）：DoS>90天
- **Stranded**（滞留）：无在售信息
- **Unsellable**（不可售）：损坏/过期

### 三、FBA费用结构与2026年变更

**FBA核心费用项目：**
- 月度仓储费（Storage Fee）
- 配送费（Fulfillment Fee）
- 移除/弃置费（Removal/Disposal Fee）
- 退货处理费（Return Processing Fee）
- 计划外服务费（Unplanned Service Fee）

**2026年FBA费用主要变更：**
1. **仓储费上调**：非旺季上调约3-5%；旺季（10-12月）上调约8-10%
2. **配送费微涨**：平均上涨2-3%（因应燃油和人工成本）
3. **长期仓储费调整**：271-365天库存从$6.90/cuft提高到$9.90/cuft
4. **库存移除费调整**：标准商品移除费从$0.50-$0.60提高到$0.60-$0.75

**仓储利用箱（Storage Utilisation Surcharge）新规（2024/2025引入）：**
- 亚马逊对高利用率仓库征收附加费
- 利用率超过50%时开始征收
- 超70%时附加费显著增加
- 这推动了卖家分散库存到不同仓库

### 四、FBA库存降本优化策略

**库存构型优化（Inventory Sku Configuration）：**
- 大件 vs 标准件 vs 小件：费用差异巨大
- 大件商品仓储费按体积计费，可能不划算
- 优化包装尺寸以匹配更低的体积重

**库存放置策略（Inventory Placement）：**
- 分散库存到多个仓库降低单一仓库爆仓风险
- 使用亚马逊库存配置服务（Inventory Placement Service）加快入仓
- 但注意：配置服务收费 vs 分散入仓的物流成本平衡

**FBA价值修复选项（Inventory Value Recovery）：**
1. **FBA Liquidations**（批量清仓）：
   - 亚马逊以原价的5-10%收购滞销库存
   - 比弃置好（弃置费$0.50-$0.60/件）
   - 适合滞销但有一定价值的商品
   
2. **Remove Order（移除订单）**：
   - 移除到海外仓或第三方仓库
   - 费用：$0.50-$0.75/标准件
   - 可用于重新贴标再销售

3. **Offers/Deals**：
   - 主动降价清理滞销库存
   - 配合LD/DOTD快速清仓

4. **退货处理**：
   - 对于不可售库存，选择清算或移除
   - 评估残余价值 vs 移除费用

### 五、库存预警与补货策略

**安全库存公式：**
```
安全库存 = (日均销量 × 备货天数) + 安全缓冲
```
- 备货天数：生产时间 + 运输时间 + 入仓时间
- 安全缓冲：通常为日均销量的20-30%

**补货优先级矩阵：**
| 优先级 | DoS | 策略 |
|--------|-----|------|
| P0（紧急） | <7天 | 紧急补货，空运 |
| P1（高） | 7-14天 | 海运补货，加快入仓 |
| P2（中） | 14-30天 | 正常补货节奏 |
| P3（低） | >30天 | 暂缓补货 |

**ABC分类补货法：**
- **A类**（高价值/高销量）：重点监控，保持60天安全库存
- **B类**（中等）：常规监控，45天库存
- **C类**（低价值/慢销）：最低库存，减少资金占用

### 六、季节性库存调整

**季节性产品补货策略：**
- 旺季前60-90天开始备货
- 使用历史数据预测旺季需求（参考去年同期）
- 不要过早备货（仓储费累积）
- 不要过晚备货（断货风险）

**旺季库存规划时间线：**
```
Q4（10-12月）：
- 7月：旺季需求预测
- 8月：开始备货
- 9月：货物陆续到仓
- 10月：全力销售，补货
- 11月：Prime Day + 黑五，网一
- 12月：圣诞节前最后补货窗口
```
