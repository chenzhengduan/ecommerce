## 学习内容摘要

### 一、供应商绩效管理

#### 1. 供应商评估维度
- **质量（Quality）**：来料合格率、抽检不良率、退货率
- **交期（Delivery）**：准时交货率、平均交期、紧急订单响应能力
- **成本（Cost）**：价格竞争力、涨价频率、成本下降空间
- **服务（Service）**：沟通响应速度、问题解决能力、配合度

#### 2. 核心KPI指标
| 指标 | 计算方式 | 优质标准 |
|------|---------|---------|
| 来料合格率（IQC Pass Rate） | 合格数量/总来料数量 | ≥98% |
| 准时交货率（OTD） | 准时交货批次/总批次 | ≥95% |
| 批次不良率 | 不良批次/总批次 | <2% |
| 交期缩短率 | （上期交期-本期交期）/上期交期 | 逐年改善 |
| 价格竞争力 | 市场调研价格 vs 实际采购价 | 处于市场较低区间 |

#### 3. 供应商分级体系
```
A类供应商（战略供应商）
- 年采购额>100万，年均增长>30%
- 质量/交期/配合度均优秀
- 给予优先付款、长期订单承诺、联合研发

B类供应商（合格供应商）
- 年采购额30-100万，质量稳定
- 常规合作，定期绩效复盘
- 支持备选供应商开发

C类供应商（观察供应商）
- 绩效指标低于标准
- 给予3-6个月改进期
- 期间同步开发备选供应商

D类供应商（淘汰供应商）
- 连续2个季度绩效不合格
- 启动切换流程，逐步减少订单
```

### 二、质检流程体系（IQC-IPQC-FQC-OQC）

#### 1. IQC（来料检验）
- **检验依据**：产品规格书、样品封样、工程图纸
- **抽样标准**：GB/T 2828.1（正常检验二级抽样方案）
- **检验项目**：外观、尺寸、功能、包装、标签
- **判定标准**：
  - A类缺陷（严重）→ 批次退货
  - B类缺陷（轻微）→ 挑拣后使用
  - C类缺陷（轻微）→ 记录但不处理

#### 2. IPQC（过程检验）
- **时机**：生产过程中关键工序节点
- **重点**：首件确认、工序流转检验
- **目的**：防止批量不良发生

#### 3. FQC（最终检验）
- **时机**：产品完工后、入库前
- **方式**：全检或抽检（根据产品风险等级）
- **内容**：功能测试、老化测试（电子产品）、模拟运输测试

#### 4. OQC（出货检验）
- **时机**：出货前最后一环
- **内容**：外箱检查、唛头确认、数量清点、装柜监督
- **标准**：参照客户验收标准或出口检验AQL

#### 5. 质检三不原则
- **不接收**：不接受不合格原材料
- **不制造**：不制造不良品
- **不流出**：不让不良品流入下一环节

### 三、交期管理

#### 1. 交期组成
```
总交期 = 备料周期 + 生产周期 + 质检周期 + 运输周期 + 缓冲时间
```

#### 2. 备料周期管理
- 常用物料安全库存（2-4周用量）
- 长交期物料（芯片/特殊五金）提前30-60天下单
- 建立物料交期预警机制

#### 3. 生产周期控制
- 明确标准产能（pcs/天）和当前排期状态
- 紧急订单处理流程（绿色通道+加急费）
- 产能预警：排期超过产能80%时提前预警

#### 4. 运输周期参考
| 运输方式 | 中国→美国 | 中国→欧洲 | 中国→东南亚 |
|---------|---------|---------|---------|
| 空运 | 7-10天 | 5-7天 | 3-5天 |
| 海运（散货） | 25-35天 | 20-28天 | 7-12天 |
| 海运（FCL） | 20-28天 | 18-25天 | 7-10天 |
| 铁路（中欧） | - | 18-22天 | - |

### 四、库存周转优化

#### 1. 库存ABC分类
```
A类（高价值，20%SKU，占80%价值）
- 盘点周期：每周
- 安全库存：较高（2-4周）
- 补货频率：高频

B类（中价值，30%SKU，占15%价值）
- 盘点周期：每月
- 安全库存：中等（1-2周）
- 补货频率：中频

C类（低价值，50%SKU，占5%价值）
- 盘点周期：每季度
- 安全库存：低（0.5-1周）或JIT
- 补货频率：低频
```

#### 2. 库存周转率计算
```
库存周转率 = 年度销售成本 / 平均库存
库存周转天数 = 365 / 库存周转率

参考标准：
- 快速消费品（FMCG）：30-45天
- 电子产品：60-90天
- 工业品：90-120天
- 服装（季节性）：季节内7-10次轮换
```

#### 3. 降库存策略
- **季节性清仓**：季末前30天开始清仓
- **捆绑销售**：与热销品捆绑快速出清
- **站内Deal**：亚马逊LD/BD快速出量
- **站外Deal**：Slickdeals/Dealabs清仓
- **FBA库存冗余处理**：FBA Liquidation（清货价值约为售价的5-10%）

### 五、仓库自动化

#### 1. 自动化层级
```
Level 1：基础仓储（手动操作，Excel管理）
Level 2：系统辅助（WMS系统，条码扫描）
Level 3：半自动化（RF终端，电子拣货，输送线）
Level 4：高度自动化（AS/RS立体仓，AGV小车，货到人系统）
Level 5：智能仓储（AI调度，数字孪生，全流程无人化）
```

#### 2. 中小卖家仓库自动化路径
- **阶段一**：WMS系统上线（店小秘/聚水潭/万里牛）
- **阶段二**：条码管理（扫码枪+标签打印机）
- **阶段三**：PDA手持终端
- **阶段四**：流水线+电子标签（Put-to-light）
- **阶段五**：AGV/AMR智能搬运

#### 3. FBA仓库布局原则
- **热销品靠近打包区**：缩短拣货路径
- **同款同色同码相邻**：提高拣货效率
- **季节品预留动态区域**：避免固定区域浪费
- **通道宽度匹配叉车/手推车**：安全且高效

### 六、分仓策略

#### 1. 分仓决策逻辑
```
考虑因素：
1. 销量分布（各站点/各仓销量占比）
2. 物流成本（头程分摊+尾程配送）
3. 配送时效（客户等待预期）
4. 仓储成本（各仓租金差异）
5. 库存集中风险（单仓故障影响）
```

#### 2. 亚马逊分仓策略
- **FBA分布式库存**（Inventory Distribution）：平台自动分仓
- **FBA库存配置服务**（Inventory Placement Service）：按件收费，单件发往更优仓库
- **建议**：对于SKU多、销量分散的卖家，使用FBA Send to Amazon（STBA）提前分配

#### 3. 海外仓分仓逻辑
- **美西仓**（LA/ Oakland）：辐射加州、内华达、亚利桑那，墨西哥
- **美东仓**（NJ/Atlanta）：辐射东海岸、五大湖、佛罗里达
- **美中仓**（TX/Chicago）：辐射中南部、覆盖全美性价比高

### 七、供应链弹性与风险管理

#### 1. 供应链风险识别
```
外部风险：
- 地缘政治（关税政策/贸易战）
- 自然灾害（台风/地震/疫情）
- 汇率波动（人民币升值/贬值）
- 物流瓶颈（港口拥堵/船期延误）

内部风险：
- 单一供应商依赖
- 质量不稳定
- 交期延误
- 产能不足
```

#### 2. 供应链弹性建设
- **供应商冗余**：核心物料2+家供应商（主供应商80%+辅供应商20%）
- **地理冗余**：不把鸡蛋放在一个地区
- **库存缓冲**：安全库存+战略库存储备
- **替代方案**：备选物流渠道、备选运输方式

#### 3. 危机应对SOP
```
Step 1：风险识别（提前30天识别）
Step 2：预案准备（备选供应商/渠道激活）
Step 3：快速决策（48小时内启动预案）
Step 4：沟通协调（客户/平台/物流商同步）
Step 5：复盘改进（事后72小时内复盘）
```

#### 4. 供应商预警指标
| 预警级别 | 触发条件 | 应对措施 |
|---------|---------|---------|
| 绿色（正常） | 无异常 | 正常跟踪 |
| 黄色（关注） | 交期延误3-5天/质量波动 | 催促+要求改善报告 |
| 橙色（警告） | 交期延误5-10天/质量问题 | 启动备选供应商+质量专案 |
| 红色（紧急） | 交期延误>10天/重大质量问题 | 紧急切换+高层介入+索赔 |

### 八、供应商开发流程

#### 1. 供应商筛选流程
```
需求确认 → 市场调研 → 初筛名单 → 资质审核 → 样品评估 → 验厂审核 → 商务谈判 → 正式导入
```

#### 2. 供应商实地验厂要点
- 生产规模（人数/设备数量/产能）
- 质量管理体系（ISO 9001/ISO 14001）
- 生产环境（5S执行/设备维护/现场管理）
- 样品质量（与封样一致性）
- 配合度（沟通/响应/改进意愿）

#### 3. 新供应商导入周期
- 样品确认：1-2周
- 小批量试单：2-4周
- 正式订单：第1个月30%
- 稳定合作：第3个月后逐步加量

## 关键收获

1. **供应商分级管理**：A/B/C/D分级+定期复盘，聚焦资源在优质供应商
2. **质检流程IQC→IPQC→FQC→OQC**：四段质检环环相扣，三不原则是核心
3. **库存ABC分类**：20%的A类SKU占80%价值，需重点管理
4. **交期=备料+生产+质检+运输+缓冲**：五段时间缺一不可
5. **供应链弹性**：核心物料2+供应商+安全库存是基本保障
6. **仓库自动化路径**：WMS→条码→PDA→电子标签→AGV，分阶段推进
