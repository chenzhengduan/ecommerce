# 亚马逊选品之产品退出策略与市场退出方法

**主题序号**: 242
**学习时间**: 2026-04-28 23:24
**学习来源**: 知识库

## 学习内容摘要

### 1. 产品退出时机判断

当产品出现以下情况时，考虑退出市场：
- **利润率持续下滑**：连续3个月利润率低于15%
- **BSR排名持续下降**：30天内下降超过50%
- **竞品价格战**：无法承受持续降价
- **供应链问题**：供应商停止供货/原材料成本上涨超30%
- **平台政策风险**：类目审核加强/合规成本增加
- **品牌负面评价增多**：退货率>15%，差评率>5%

### 2. 产品退出策略

**渐进式退出**（推荐）：
1. 停止广告投放，消耗现有库存
2. 小幅涨价，测试价格敏感度
3. 减少库存补充，控制滞销风险
4. 捆绑销售或清仓促销
5. 彻底下架

**快速退出**：
1. 直接清仓促销（Lightning Deal/7-Day Deal）
2. 站外Deal网站引流清库存
3. 批量折扣批发给分销商
4. 捐赠或销毁（考虑仓储成本vs残值）

### 3. 库存处理方案

- **FBA库存**：通过 Outrageous Deal / 亚马逊清仓 / 站外促销
- **海外仓库存**：分销给其他卖家 / 退运回国 / 就地销毁
- **国内库存**：分批销售 / 捐赠 / 报废处理

### 4. 数据保留与复盘

退出前需记录：
- 销量趋势数据
- 利润变化曲线
- 竞品价格走势
- 用户评价分析
- 失败原因总结

## 判断正确性

✓ 核心框架正确，符合亚马逊运营逻辑
✓ 渐进式退出是常见最佳实践
✓ 库存处理方案具有实用性
