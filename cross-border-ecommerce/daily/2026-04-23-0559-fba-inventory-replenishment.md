# 跨境电商学习笔记 — 2026-04-23 05:59

## 主题：FBA库存管理与补货策略

---

### 一、学习内容摘要

#### 1. FBA库存管理的核心指标

**关键指标：**
- **IOP (Inventory Performance Index)**：亚马逊库存绩效指标，满分1000分，400分以上为良好
- **Sell-through Rate**（售出率）= 销量 / 平均库存量
- **Stockout Rate**（缺货率）：断货对排名影响极大
- **Days of Supply**（供应天数）：当前库存预计可支撑天数

**库存健康标准：**
- 非季节性产品：保持30-60天库存
- 季节性产品：旺季前提前2-3个月备货
- 高周转产品：保持安全库存7-14天

#### 2. 补货策略类型

**定量补货（Reorder Point Method）**
```
Reorder Point = 平均日销量 × 备货周期 + 安全库存
```
- 适合：稳定销售产品
- 优点：简单易执行
- 缺点：无法应对突然的销量波动

**定期补货（Periodic Review Method）**
- 每周/每两周固定检查库存
- 适合：多SKU管理
- 优点：批量处理效率高
- 缺点：可能错过最佳补货时机

**动态补货（Dynamic Replenishment）**
- 结合历史数据、销售预测、竞品动态自动计算
- 使用补货工具：RestockPro、Inventory Manager等
- 适合：大SKU数量、多平台运营

#### 3. 备货周期与物流选择

**海运 vs 空运的选择逻辑：**
- 海运：时效25-40天，低成本，适合稳定销售的爆款
- 空运：时效7-15天，高成本，适合新品测试或紧急补货
- 海外仓：平衡时效与成本

**备货量计算公式：**
```
备货量 = 日均销量 × (海运备货周期 + 安全天数) × 1.2(保险系数)
```

#### 4. 库存成本优化

**降低FBA库存成本的策略：**
1. **减少冗余库存**：通过促销、捆绑销售清理滞销SKU
2. **优化包装尺寸**：减少包装体积可降低FBA仓储费
3. **合理使用AGL**：亚马逊全球物流可能比第三方便宜
4. **长期仓储费规避**：在6月和12月清理超365天的库存
5. **inventory placement service**：选择性发送可节省运费

**库存预测工具：**
- Amazon库存看板（Inventory Dashboard）
- 第三方工具：Helium10、Sellics、Inventory Protector
- 自建Excel/BI系统：基于历史数据预测

#### 5. FBA容量管理（2024年新规）

亚马逊对FBA容量实施限制：
- 使用容量配额系统管理库存
- 通过 Seller Central 申请额外容量
- 预留容量用于季节性高峰
- 季度性评估和调整

#### 6. 断货应对策略

**断货后的处理步骤：**
1. 立即降低广告竞价，减少无效支出
2. 发送FBA紧急补货（空运）
3. 暂停优惠券和促销活动（避免差评）
4. 跟卖自己的FBA库存（如果有）
5. 断货后不要大幅调价，等库存恢复

### 二、对不对的判断

✅ **正确理解的部分：**
- 补货公式和逻辑基本正确
- IOP指标是亚马逊真实使用的指标
- 安全库存概念合理
- 海运vs空运的选择标准正确

⚠️ **需要补充或调整的部分：**
- IOP现在改名为IPI（Inventory Performance Index），且分数已从1000改为100
- 亚马逊容量限制的具体数值经常调整
- 补货公式中的1.2保险系数需要根据产品稳定性调整
