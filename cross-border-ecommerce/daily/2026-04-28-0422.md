## 一、核心概念

### 1.1 销售漏斗（Sales Funnel）

销售漏斗是描述顾客从认知到购买全过程的层次化模型。亚马逊场景下的典型漏斗：

```
曝光 → 点击 → 浏览 → 加购 → 购买 → 复购
  ↓     ↓      ↓      ↓      ↓
 曝光量  CTR   CR    CAR    转化率 复购率
```

**各层级指标**：
- **曝光量（Impressions）**：listing 展示给顾客的次数，由搜索排名和广告投放决定
- **点击率 CTR** = 点击数 / 曝光量，反映 listing 吸引力（主图、标题、价格）
- **浏览率（Page Views Rate）** = 浏览详情页次数 / 点击数，反映详情页内容质量
- **加购率（Add to Cart Rate）** = 加购数 / 浏览量，反映产品价值和对比竞争力
- **购买转化率（Conversion Rate）** = 订单数 / 流量，反映价格和整体竞争力
- **复购率**：重复购买顾客占比，反映产品满意度和品牌忠诚度

### 1.2 亚马逊漏斗特殊点

1. **搜索漏斗为王**：大部分流量来自搜索，A9 算法决定曝光
2. **广告漏斗独立**：Sponsored 产品有独立曝光-点击漏斗
3. **BSR 排名**：每小时更新，本质是销量转化排名
4. **Buy Box 漏斗**：购物车归属决定转化归属

## 二、转化率优化策略

### 2.1 Listing 转化优化（曝光→点击→浏览）

| 要素 | 优化点 | 目标指标 |
|------|--------|----------|
| 主图 | 白底、高清、场景化、核心卖点可视化 | CTR > 0.5% |
| 标题 | 品牌 + 核心卖点 + 关键特征 + 适用场景 | 搜索相关性 |
| 价格 | 参考竞品、动态定价、优惠券 | 价格竞争力 |
| Bullet Points | 5 点核心卖点，场景化表达 | 浏览时长 |
| 图片 | 主图 + -infographic + 场景图 + 尺寸图 + 视频 | 页面停留 |

### 2.2 详情页优化（浏览→加购→购买）

- **A+ 内容**：品牌故事、产品对比图表、FAQ
- **QA 板块**：真实问题和解答，增加信任
- **评论**：引导好评，管理差评
- **优惠券**：刺激加购和转化
- **促销**：Prime 专享、Deal、LD

### 2.3 流量漏斗协同

- **站外流量**：Facebook/Google广告 → landing page → 亚马逊
- **站内流量**：关联推荐 + FBT（Frequently Bought Together）
- **品牌流量**：品牌旗舰店 → 产品系列

## 三、关键指标计算

### 3.1 核心漏斗指标

```
整体转化率 = 订单数 / 曝光量
广告转化率 = 订单数 / 广告点击数
自然搜索转化率 = 自然订单数 / 自然流量
详情页转化率 = 订单数 / 详情页浏览量
```

### 3.2 归因分析

- **首触归因**：曝光功劳归给首次触点
- **末触归因**：转化功劳归给最后点击
- **线性归因**：各触点均分功劳
- **时间衰减归因**：越接近转化的时间触点贡献越大
