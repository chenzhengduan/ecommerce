# 跨境电商每日学习 - 2026-04-23

## 主题：03-账号注册与店铺搭建

### 学习摘要

#### 一、主流平台账号类型

| 平台 | 账号类型 | 特点 |
|------|----------|------|
| Amazon | 专业卖家账号 / 个人卖家 | 专业卖家按月收费，适合认真做业务 |
| 速卖通 | 企业账号 / 个人账号 | 更适合轻量化起步 |
|Shopify | 基础/月度订阅 | SaaS建站，月付$29起 |
| TikTok Shop | 跨境卖家 / 本地卖家 | 东南亚/欧美市场快速起量 |

#### 二、Amazon账号注册核心要点

1. **公司注册**
   - Amazon大多数站点需要企业营业执照（北美站个人也可以）
   - 建议注册有限责任公司，便于后续账号申诉和税务合规
   - 注册地址建议与实际运营地址一致（可使用虚拟地址服务）

2. **资料准备**
   - 营业执照（真实）
   - 法人身份证
   - 双币信用卡（Visa/MasterCard）
   - 第三方收款账户（派安盈/万里汇/连连支付等）
   - 电话号码（未注册过Amazon）

3. **账号类型选择**
   - **Individual（个人）**：按件收费，适合测款，不适合长期运营
   - **Professional（专业）**：月费$39.99，适合认真做，FBA必备

4. **二审（KYC）预防**
   - 提前准备：真实经营地址、账单（水电煤）
   - 避免VPN/IP频繁变动
   - 信用卡和营业执照主体一致

#### 三、Shopify建站基础流程

1. **注册阶段**
   - 使用邮箱注册（避免用QQ/163等国内邮箱，用Gmail/企业邮箱）
   - 域名建议：品牌名+.com，短小好记
   - Shopify提供免费域名迁移或购买

2. **基础配置**
   - 店铺政策：退换货政策、隐私政策、服务条款（必须）
   - 运费设置：按重量/地区计算或包邮
   - 支付网关：Shopify Payments / Stripe / PayPal

3. **必备Apps推荐**
   - Oberlo（速卖通dropshipping选品）
   - Privy（弃单挽回弹窗）
   - Loox（评价图片展示）
   - Klaviyo（邮件自动化）

#### 四、账号安全基础

1. **登录环境**
   - 固定IP登录，避免账号关联（同一IP登录多个账号=关联风险）
   - 浏览器指纹隔离（使用Chromium多实例或专门的指纹浏览器如Linkletter）
   - 推荐使用云服务器VPS固定IP

2. **二步验证**
   - 必须开启Google Authenticator二次验证
   - 备份验证码保存在安全位置

3. **账号关联风险**
   - 不同账号间禁止使用相同资料（信用卡、收款账户、IP地址）
   - 关联账号被封，可能波及其他账号
