## 一、轻小商品计划（Amazon Small and Light）核心机制

### 什么是轻小商品计划？
亚马逊轻小商品计划（Small and Light）是亚马逊针对重量轻、体积小、售价低的商品推出的特殊物流解决方案。这些商品享受更低的FBA配送费用，适合低价商品（如厨房小工具、饰品、美妆配件、办公文具等）提升利润率。

### 适用条件（以美国站为例）
| 条件 | 具体要求 |
|------|---------|
| 重量 | ≤ 10 oz（约283g），部分类目上限12 oz |
| 尺寸 | 标准尺寸，最长边 ≤ 15英寸，最长边+周长 ≤ 23英寸 |
| 价格 | ≤ $15（通常要求 $7-$15 之间） |
| 品类 | 并非所有品类都适合，需参考亚马逊类目要求 |
| 存储费 | 无月度存储费（美国站），但有旺季附加费 |

### 费用结构（美国站参考，2024年数据）

**轻小商品配送费（Standard Size）：**
- 首仓配送费：约 $2.50 - $3.00（比标准FBA低约30-40%）
- 重量分段更简单：≤2 oz / 2-4 oz / 4-6 oz / 6-8 oz / 8-10 oz 等

**相比标准FBA的费用节省示例：**
| 商品重量 | 标准FBA配送费 | 轻小商品配送费 | 节省比例 |
|---------|--------------|--------------|---------|
| 4 oz | ~$4.50 | ~$2.80 | ~38% |
| 8 oz | ~$5.50 | ~$3.30 | ~40% |

### 核心优势
1. **配送费大幅降低**：适合低价商品维持利润空间
2. **无月度存储费**：美国站轻小商品不收月度存储费（旺季除外）
3. **配送速度快**：仍享受Prime资格，通常3-5个工作日
4. **自动适用**：符合条件的ASIN亚马逊会自动加入，无需额外申请

### 潜在限制
1. **价格竞争激烈**：轻小商品市场通常竞争激烈，需有差异化
2. **退货成本高**：低价商品退货处理成本相对收益可能不划算
3. **适用类目有限**：并非所有品类都符合轻小计划要求
4. **重量严格限制**：超过10 oz的商品无法参与
5. **无批量处理费优惠**：某些场景下批量处理费可能比标准FBA更高

## 二、亚马逊SFP（Seller Fulfilled Prime）方案

### SFP基本概念
Seller Fulfilled Prime (SFP) 允许卖家自配送商品获得Prime标志，但必须满足亚马逊的绩效要求。适合有自身仓储和物流能力的卖家。

### SFP核心要求
1. **配送绩效**：99%以上的追踪率/OPS (On-Time Performance) ≥ 97%
2. **配送地区**：仅限美国本土48州
3. **配送时效**：周六/周日也需当日/次日达
4. **退货政策**：必须提供Pre-paid Return Labels
5. **品类限制**：部分品类不适用SFP

### SFP vs FBA对比

| 对比维度 | FBA | SFP |
|---------|-----|-----|
| Prime标志 | ✅ 自动 | ✅ 满足条件后 |
| 配送费用 | 较高 | 可控（自控） |
| 库存管理 | 亚马逊处理 | 卖家自行管理 |
| 退货处理 | 亚马逊处理 | 卖家处理 |
| 适用范围 | 全品类 | 部分受限 |
| 门槛 | 仓储费 | 绩效要求严格 |

### 适合SFP的卖家类型
1. 本地有仓储和物流团队（中大件商品优势明显）
2. 商品重量>10 oz超出轻小限制但仍适合自配送
3. 需要控制物流成本的大件商品卖家

## 三、低成本物流方案选择框架

### 物流方案对比矩阵

| 方案 | 适用场景 | 成本 | 时效 | 适合商品类型 |
|------|---------|------|------|------------|
| FBA轻小 | 低价轻小商品 | 低 | 3-5天 | <10oz低价商品 |
| FBA标准 | 常规商品 | 中 | 2-3天 | 中等重量商品 |
| FBA海外仓 | 欧洲/多国 | 中高 | 5-10天 | 跨境欧洲商品 |
| 第三方海外仓 | 灵活存储 | 中 | 3-7天 | 多平台卖家 |
| 直邮（小包） | 测试市场 | 低 | 15-30天 | 低价值商品 |
| SFP自配送 | 本地仓卖家 | 可控 | 1-2天 | 有本地仓的中大件 |

### 低成本物流决策树

```
商品重量 ≤ 10 oz?
├── 是 → 价格 ≤ $15?
│   ├── 是 → 优先考虑 FBA轻小商品计划
│   └── 否 → FBA标准 vs SFP成本对比
└── 否 → 考虑以下路径：
    ├── 有本地仓储 → SFP或FBA标准
    ├── 多平台销售 → 第三方海外仓
    └── 测试阶段 → 直邮小包
```

## 四、降低FBA成本的综合策略

### 1. 商品重量优化
- 尽量将商品控制在轻小计划的10 oz以内
- 优化产品包装：减少不必要包装材料，在合规范围内减轻重量
- 组合销售：将互补商品组合，通过整体降低物流成本

### 2. 仓储费用优化
- 合理规划旺季前的移仓时机，避免旺季高存储费
- 使用FBA Inventory Placement Service集中入仓减少内部操作费
- 定期清理超龄库存（库存费每月累积）

### 3. SKU管理优化
- 淘汰长期仓储费超过商品本身价值的SKU
- 使用FBA Liquidations回收部分价值
- 通过促销或降价清理滞销品

### 4. 物流渠道组合
- 主推产品走FBA保证时效和排名
- 长尾产品走第三方海外仓或FBM降低成本
- 测试新品阶段使用小包直邮验证市场

### 5. 谈判与规模化
- 第三方海外仓：争取量越大越优惠的仓储费
- 物流服务商：与货代谈判争取长期合作折扣
- FBA：量大的情况下关注FBA Capacity Manager配额

## 五、案例分析：轻小商品计划选品思路

### 典型适合轻小计划的商品特征
1. **重量轻**：单件重量 ≤ 10 oz（理想 ≤ 6 oz）
2. **体积小**：避免体积重超过实际重量
3. **价格适中**：$7-$15 区间最有竞争力
4. **需求稳定**：日常消耗品或礼品类（非季节性过强）
5. **竞争适度**：避开红海大类，寻找细分机会

### 参考品类方向
- **家居收纳**：塑料收纳盒、厨房小工具
- **美妆工具**：化妆刷、指甲贴、饰品
- **办公文具**：笔筒、回形针收纳、桌面装饰
- **母婴用品**：儿童玩具配件、辅食工具
- **运动户外**：瑜伽配件、骑行小装备

### 避开类型
- 易碎品（退货率高）
- 带电池商品（认证复杂）
- 液体/膏状（海关限制）
- 季节性过强（库存风险大）

## 六、总结与实操建议

### 核心要点
1. **轻小商品计划适合$7-15之间、重量<10oz的低价轻小商品**，可节省30-40%配送费
2. **SFP适合有本地仓储和物流能力的卖家**，特别是中大件商品可以控制自身配送成本
3. **物流方案选择应根据商品特征、重量、体积、价格、目标市场综合决策**
4. **降低成本的核心在于：选品阶段的重量优化+仓储管理+SKU清理+渠道组合**
