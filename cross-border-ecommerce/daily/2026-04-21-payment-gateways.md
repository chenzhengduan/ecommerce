## 一、主流支付网关概览

### 1. Stripe
- **定位：** 独立站最流行的国际支付网关
- **支持：** 信用卡（Visa/MC/Amex）、Google Pay、Apple Pay、ACH转账、Alipay、WeChat Pay
- **覆盖：** 135+国家，40+货币
- **手续费：** 2.9% + $0.30（美国），欧洲略低，国际交易加0.5-1%
- **特点：** API 最完善，开发者友好，PCI DSS Level 1 合规，实时数据面板强，支持订阅/Invoicing/CONNECT 多商户平台
- **结算：** T+2 或立即（Stripe Capital 提前放款）

### 2. PayPal
- **定位：** 全球覆盖率最高的电子钱包
- **支持：** PayPal余额、信用卡、银行转账、本地支付（部分国家）
- **手续费：** 2.99% + $0.30（美国电商），跨境额外 1.5%
- **特点：** 用户信任度高（尤其欧美），一键结账（PayPal Checkout），争议处理（Dispute/Chargeback）相对成熟
- **结算：** T+1（Instant Transfer 收 1.5% 手续费）
- **注意：** 账户审查严格，风控触发率高（尤其是新账号）

### 3. Adyen
- **定位：** 企业级一体化支付平台（IPO 公司）
- **支持：** 全球 300+ 支付方式，一站式接入
- **特点：** 面向中大型商家，支持 150+ 货币，覆盖 50+ 国家本地支付方式（Klarna/Sofort/iDEAL/Poli 等）
- **定价：** 定制化议价（量越大费率越低）
- **适合：** 月流水 > $50 万美元的商家

### 4. Braintree（PayPal 旗下）
- **定位：** 企业级信用卡处理，PayPal 生态一部分
- **特点：** 无月费，PCI 合规简化，PayPal + 信用卡一体化，Venmo 整合

### 5. Square
- **定位：** 小微商家 / 线下 + 线上整合
- **特点：** 月费 $0（在线) 或 $29（ Plus），POS 整合强，免费电商网站建站

### 6. Checkout.com
- **定位：** 欧洲市场领先的国际支付网关
- **特点：** 欧洲本地支付方式覆盖全（SEPA/Klarna/Sofort），定价透明，拒付管理强

## 三、独立站收款配置最佳实践

### 1. 标准配置（通用型独立站）
```
Stripe（主）+ PayPal（辅）+ 本地支付插件
```

**原因：**
- Stripe 手续费最低、API 最强、退款处理灵活
- PayPal 覆盖无卡用户，提供额外信任背书
- 本地支付插件（如 Stripe 的 Klarna、PayPal 的 Venmo）提升转化

### 2. 欧洲站必备
```
Stripe + Klarna + SEPA + 当地主流方式
```
欧洲 BNPL 使用率：德国 40%，瑞典 50%，荷兰 45%

### 3. 东南亚站必备
```
Stripe + 本地支付插件（如 PaySee / dLocal / PayNow）
```

### 4. Shopify 专属支付
- **Shopify Payments：** 官方集成，封号风险低，但仅部分国家支持（美/英/加/澳/德/爱/新/西/港等）
- **手续费：** 2.9% + $0.30（使用 Shopify Payments 可省去第三方手续费差价）
- **注意：** 使用 Stripe/PayPal 也可通过 Shopify Payment 以外的渠道接受付款

## 五、收款优化核心指标

### 1. 支付转化率（Payment Conversion Rate）
- **定义：** 到达支付页并成功完成支付的比例
- **行业均值：** 70-85%（国际信用卡）
- **低于 70% 需排查：** 页面加载慢、支付选项少、信任图标缺失、强制注册

### 2. 拒付率（Chargeback Rate）
- **安全线：** < 0.5%
- **危险线：** > 1%
- **影响：** 高拒付导致网关降级、封号、保证金要求提高

### 3. 结算周期 & 资金占用
- **标准 T+2 vs Instant：** Instant 收 1.5% 但解决资金占用问题
- **滚动预留（Rolling Reserve）：** 新商家通常被预留 5-10%，6-90 天释放

### 4. 支付成功率（Authorization Rate）
- **目标：** > 95%
- **失败原因：** 卡过期、额度不足、风控拦截、3DS 验证失败

## 六、降低拒付（Chargeback）实战策略

1. **3D Secure 2（3DS2）：** 欧盟强制（PSD2），将风控责任从商家转向发卡行，减少拒付
2. **清晰的退款政策：** 页面显眼位置展示，符合消费者预期
3. **订单确认邮件：** 含商品描述、配送时间、联系方式，减少"未授权"误判
4. **包装内塞退款卡：** 直接联系客服解决，比走平台更快
5. **地址验证（AVS）：** 匹配账单地址与收货地址
6. **风控规则调优：** Stripe Radar / PayPal Seller Protection 配置
7. **IP 地理位置检测：** 欺诈高发地区单独处理

## 七、支付安全与合规

### PCI DSS 合规（支付卡行业数据安全标准）
- **Level 4（大多数中小商家）：** SAQ-A（最简问卷）或 Self-Assessment Questionnaire
- **使用 Stripe/PayPal：** 本身已 PCI 兼容，商家无需持有卡数据
- **自己处理：** 必须 PCI DSS Level 1（最严），成本极高

### PSD2 & SCA（欧盟）
- **强客户认证（Strong Customer Authentication）：** 50 欧元以上交易需双重验证（密码+手机/人脸）
- **Stripe/PayPal：** 自动处理 S2A豁免（低风险交易）
- **例外：** 商家可向发卡行申请"Trusted Beneficiary"豁免

### 税务合规
- **VAT：** 欧盟 B2C 数字服务超过阈值（€10,000）需在消费国注册 VAT
- **GST：** 澳大利亚 $75,000+、加拿大 CAD $30,000+ 等类似阈值
- Stripe Tax / Quaderno 可自动计算

## 八、收款账户开通避坑

1. **注册主体一致性：** 支付账户主体与独立站主体最好一致（减少风控）
2. **银行账户独立：** 收款账户与运营成本账户分开
3. **提前准备材料：** 营业执照、网站 URL、产品类别、预计月流水
4. **避免多账户关联：** 同主体在 Stripe/PayPal 开多个账户会被关联识别
5. **资金链规划：** 新账号 2-4 周审核期，提前备好运营资金

## 学习内容摘要

独立站支付网关体系包含三个层次：**国际信用卡网关（Stripe/PayPal/Braintree）**、**本地支付方式（BNPL/Klarna/SEPA/本地钱包）**、**区域专用平台（Adyen/Checkout/dLocal）**。Stripe 是中小独立站首选（手续费低、API 强），PayPal 提供信任背书，欧洲市场必须接入 Klarna 等 BNPL，东南亚需接入当地钱包。核心优化目标是**提高支付转化率（目标 85%+）**和**压低拒付率（<0.5%）**，通过 3DS2、AVS、Radar 风控规则等手段实现。
