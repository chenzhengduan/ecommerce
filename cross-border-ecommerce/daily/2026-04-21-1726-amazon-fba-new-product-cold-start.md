## 一、新品扶持期窗口：亚马逊给新品的黄金时间

### 扶持期时长
- **标准扶持期**：上架后前90天（部分类目60天）
- **核心流量倾斜**：亚马逊对新品的关键词排名、类目排名给予初始加权
- **BD（Best Deal）豁免**：新品满足条件可优先获得Deal推荐资格
- **黄金法则**：前30天定生死，前90天定江山

### 扶持期本质
亚马逊通过算法识别"新品"，给予额外的曝光机会。卖家的任务是**在扶持期内把自然排名推上去**，否则扶持结束后流量会明显下滑。

## 二、上架前的准备工作（测款 + Listing基础）

### 1. 测款阶段（冷启动关键）
| 测款方式 | 适合产品 | 优缺点 |
|---------|---------|--------|
| Amazon Posts测款 | 有一定粉丝基础的品牌 | 免费，但数据有限 |
| 社交广告预热 | 有社媒粉丝积累 | 需要测试预算 |
| 亚马逊小额试销 | 有成熟竞品可对标 | 最真实市场反馈 |
| 亚马逊VC/PR样品 | 高客单价/定制产品 | 零库存风险 |

**测款核心指标**：点击率（CTR > 0.5%）、转化率（CVR > 10%）、加购率

### 2. Listing上架Checklist
- [ ] 标题：核心关键词 + 差异化卖点 + 品牌名（前60字符最重要）
- [ ] 五点描述：痛点 + 解决方案 + 产品特点 + 使用场景 + 售后保障
- [ ] 产品图片：白底主图 + 场景图 + 尺寸图 + 细节图 + 包装图（A+进阶）
- [ ] A+页面：第一屏Banner + 对比图表 + 场景图 + 细节图 + 品牌故事
- [ ] 搜索词：250字节内填满，按相关性和搜索量排序
- [ ] 定价：参考竞品价格 + 考虑Coupon/Promotion空间
- [ ] 库存：首批备货量 = 预计日销 × 安全库存天数（30~45天）
- [ ] 变体：若有多个变体，新品期不要全部上架，先测主推变体

## 三、广告冷启动：如何快速积累数据

### 1. 广告架构设计
```
新品期广告架构（推荐）：
├── 自动广告（开启广泛匹配，快速积累关键词数据）
├── 手动关键词广告（精准匹配核心长尾词）
└── 展示型广告（再营销，提高品牌曝光）
```

### 2. 竞价策略
| 阶段 | 策略 | 目的 |
|------|------|------|
| 第1~2周 | 竞价略高于建议价 + 开自动广告 | 获取数据，养关键词 |
| 第3~4周 | 根据数据调整，重点打转化好的词 | 降低ACOS，提升排名 |
| 第5周后 | 稳定竞价，观察自然排名变化 | 逐步降低广告依赖 |

### 3. 预算分配
- 新品期：广告预算占总销售额的15~30%（高于成熟期）
- 核心原则：**在扶持期内把广告数据跑满**，积累足够的点击和转化信号

### 4. 竞价调整节奏
- 每3~5天分析一次广告数据
- 将点击>10次但无转化的关键词加入否定
- 将ACOS<30%的优质关键词提高竞价

## 四、评价积累：新品期最重要的运营动作

### 1. Vine计划（评价获取利器）
- **费用**：$200~$250/ASIN（亚马逊官方）
- **效果**：通常获得3~5条带Vine标记的高质量评价
- **时机**：上架后尽快注册，越早越好
- **条件**：需要品牌备案，真实产品

### 2. 早期评论人计划
- **费用**：$60~$250/SKU（按评论数量阶梯收费）
- **适合**：未注册Vine的新品
- **注意**：亚马逊已逐步用Vine替代该计划

### 3. 催评策略
| 时机 | 方式 |
|------|------|
| 签收后3~5天 | 请求评论（Request Review按钮）|
| 差评出现24小时内 | 主动联系客户解决问题 |
| 好评买家 | 邀请留下评论（Amazon's "Request a Review"）|

### 4. 评价数量目标（参考值）
| 类目竞争度 | 新品期30天目标 |
|-----------|--------------|
| 低竞争 | 5~10条 |
| 中等竞争 | 10~20条 |
| 高竞争（红海）| 20~30条 |

## 五、库存管理：避免断货与积压

### 1. 新品期库存策略
- **首批备货**：宁多勿少（FBA库存不足会严重影响排名）
- **补货公式**：安全库存 = 日均销量 × 备货周期（海运30~40天）
- **断货应对**：若预判断货，提前涨价 + 减少广告预算（降低出单速度）

### 2. 移仓策略（降本）
- 从FBA移仓到第三方海外仓：用海外仓发货降低FBA仓储费
- 注意：移仓期间无Prime标识，转化率会下降

### 3. 库龄管理
- FBA长期仓储费：超过365天收取$6.90/cu ft
- 避免积压：新品期结束后（90天）仍未起量的产品及时清仓

## 六、定价策略：如何在新品期建立价格优势

### 1. 常见新品定价法
| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 竞争定价法 | 参考BSR前10价格，定价低5~10% | 有强竞品对标 |
| 渗透定价法 | 低于成本价切入，快速抢排名 | 资金充足，有爆款潜力 |
| 价值定价法 | 根据产品差异化价值定价 | 有明显差异化卖点 |

### 2. 促销活动组合
- **Coupon**：5%~15%折扣，吸引价格敏感型买家
- **Prime专享折扣**：针对Prime会员，通常5%起
- **LD/BD**：若有资格申请，可快速提升排名

### 3. 定价调整节奏
```
新品期定价路径：
第1~2个月：低价切入（快速积累销量和评价）
第3个月：逐步提价（每次提价不超过10%，观察转化率）
第4个月后：稳定在目标价格（利润优先）
```

## 七、常见错误与避坑指南

### 新品期最常见的5个错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 上架前未做竞品分析 | 定价错误，转化率低 | 先分析BSR前10的价格、评分、关键词 |
| 广告架构混乱 | ACOS过高，浪费预算 | 先自动广告测词，再开手动 |
| 库存备货不足 | 断货后排名下滑 | 首批备货量 = 日销 × 60 |
| 不关注差评 | 评分低于4.0难以出量 | 差评24小时内联系客户解决 |
| 盲目追求ACOS低 | 放弃抢占流量 | 新品期ACOS可接受30~50%，看ROAS |

### 新品期运营红线
1. ❌ 不要在上架前大量备货（先小额测款）
2. ❌ 不要一上架就打大额Coupon（容易吸引撸羊毛的买家）
3. ❌ 不要忽视差评（4.0以下评分会严重影响转化）
4. ❌ 不要在数据不足时频繁调整Listing（亚马逊需要数据积累）
5. ❌ 不要断货（断货超过3天，自然排名大幅下滑）

## 学习内容摘要

亚马逊FBA新品冷启动是一个系统工程，核心是**在90天扶持期内完成以下任务**：
1. **Listing优化**：标题/五点/图片/A+基础打牢
2. **广告冷启动**：自动广告测词→手动广告打转化词，预算充足
3. **评价积累**：Vine计划 + 早期评论人 + 主动催评，目标4.3+评分
4. **库存管理**：首批备货充足，补货及时，避免断货
5. **定价策略**：低价切入→逐步提价→稳定利润

新品期本质是**花钱买数据和排名**的过程，不要期望一开始就盈利。扶持期结束后（90天），若自然排名已稳定，广告ACOS会自然下降，进入盈利阶段。
