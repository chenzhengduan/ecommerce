# 51-亚马逊Coupon设置与品牌促销

**学习时间**：2026-04-27 11:50

## 学习内容摘要

### Coupon（优惠券）设置

1. **创建入口**：卖家中心 → 广告 → 优惠券 或 Listing页面直接设置
2. **设置要求**：需要Prime资格或至少有评级
3. **折扣设置**：可设置百分比折扣或固定金额折扣
4. **预算控制**：每日预算上限，避免超支
5. **人群定向**：可定向特定人群（如复购客户）

### 品牌促销（Brand Coupon）

1. **品牌备案要求**：需要完成亚马逊品牌备案
2. **促销类型**：品牌优惠券、会员专享折扣、买一赠一等
3. **叠加规则**：可与LD/7DD等Deal叠加
4. **预算限制**：有预算消耗机制

### 最佳实践

- 新品期设置较高折扣（如20%+）快速获取评价
- 结合站内广告投放效果更好
- 监控ACOS，避免长期高折扣导致亏损
- 可设置渐进式折扣（第一周30%，逐渐降低）
