## 一、亚马逊Vine计划详解

### 什么是Vine计划
Amazon Vine是亚马逊官方产品评论项目，邀请平台最可信的评测者"Vine Voices"免费获取产品并发布真实评测。这是获取早期评论的核心正规渠道。

### Vine Voices的选拔标准
- 基于reviewer rank（评测排名）
- 排名反映评测质量和帮助性
- 由其他亚马逊用户评判选出
- 仅限亚马逊最受信赖的评测者

### 入围要求（卖家端）
1. 必须是品牌备案卖家（Brand Registry）
2. 产品必须通过FBA配送，状态为"New"
3. Listing评论数少于30条
4. 产品不得是"成人用品"
5. 产品必须已上架（已发布）
6. 需确保有可用库存
7. Listing需有图片和描述

### 费用结构（2023年10月19日更新）
| 注册数量 | 费用 |
|---------|------|
| 1-2件 | $0 |
| 3-10件 | $75 |
| 11-30件 | $200 |

计费规则：首条评论发布后7天收费，按注册数量档位一次性收取。

### 注册上限
- 同时最多注册**200个ASIN**
- 单个ASIN最多获得**30条**Vine评论
- 已有5条评论的Listing，最多再获得25条
- 达到30条评论后自动退出计划

### 流程与时效
1. Seller Central → Advertising → Vine → 开始注册
2. 提交ASIN并确认注册
3. 前28天：仅定向展示给同品类的Vine Voices
4. 28天后若仍有库存：开放给全部Vine Voices
5. 评测者收到产品后需4-8周撰写评测
6. **无强制评测义务**——评测者无义务必须写评论

### 重要风险提示
- **不保证好评**：Vine Voices以诚实著称，发现问题会直接批评
- **无法删除评论**：除非违反亚马逊政策，否则无法移除差评
- **产品一致性**：Listing描述与实物不符会被评测者重点指出

## 二、其他获取早期评论的方法

| 方法 | 说明 |
|------|------|
| 邮件自动化请求 | 通过Jungle Scout等工具自动发送评价请求 |
| 产品插卡 | 合规范围内在包装内放置感谢卡（不可要求好评） |
| Early Reviewer Program | 已于2019年部分站点停用，现以Vine为主 |
| 请求评论按钮 | 亚马逊官方"Request a Review"功能（订单后4-30天） |

## 三、学习内容摘要

- Vine计划是亚马逊官方认可的正规获取评论渠道
- 核心价值：建立初期可信度，帮助新品突破零评论困境
- 费用$0-$200，取决于注册数量档位
- Vine Voice评测通常非常详尽，**经常附带产品图片**
- 差评风险真实存在——评测者独立且诚实
- 品牌备案是必要前提

## 五、实操建议

1. **新品上架后优先注册Vine**，快速积累可信评论
2. **确保产品质量和listing准确性**，避免Vine评测者留下差评
3. **不要只依赖Vine**，需配合其他合规评价获取方式
4. **预算有限时**，优先注册3-10件档位（$75），性价比最高
5. **避免注册已有30+评论的ASIN**，浪费费用且无意义
