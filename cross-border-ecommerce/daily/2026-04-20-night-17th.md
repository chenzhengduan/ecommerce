# 2026-04-20 学习日志

## 学习主题：亚马逊广告投放详解（PPC/AMS）

---

## 一、广告类型

### 1. SP（Sponsored Products）
- 最常见的商品推广广告
- 展现在搜索结果页和商品详情页
- 按点击付费（CPC）
- 适合：引流、提升销量、新品冷启动

### 2. SB（Sponsored Brands）
- 品牌推广广告（曾经叫Headline Search Ads）
- 可展示品牌Logo + 多款商品 + 自定义标题
- 展现在搜索结果顶部
- 适合：品牌曝光、引导发现品牌

### 3. SD（Sponsored Display）
- 展示型广告
- 可定向到ASIN或受众（看过某类目的用户）
- 适合：再营销、竞品截流

---

## 二、广告结构与Campaign设置

### Campaign（广告系列）
- 设置预算（每日/每周/每月）
- 设置竞价策略（固定/动态只下调/动态上下调整）
- 设置投放位置（仅搜索/仅详情页/搜索和详情页）
- 选择地区/语言

### Ad Group（广告组）
- 同一个广告组内共享关键词竞价
- 可批量添加关键词或ASIN定向

### 关键词匹配类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 广泛匹配 | 包含所有关键词变体 | baby shoes → baby shoes for girls |
| 词组匹配 | 包含词组顺序 | "baby shoes" → baby shoes red |
| 精确匹配 | 完全一致 | [baby shoes] → 仅 baby shoes |

---

## 三、竞价策略详解

### 1. 固定竞价（Fixed Bids）
- 所有展示统一使用设置的竞价
- 适合：稳定期、测试数据

### 2. 动态竞价-只向下（Dynamic bid - down only）
- 系统检测到转化差时，自动降低竞价
- 适合：成熟产品、利润空间有限的商品

### 3. 动态竞价-向上向下（Dynamic bids - up and down）
- 转化好时自动提价，差时降竞价
- 适合：新品测试期或高潜力产品
- 提价幅度可设置（最多+100%）

### 4. 竞价+策略（Bid+）
- 当广告位在搜索结果首页顶部时，允许在原竞价基础上额外加价
- 可设置0%~100%的加成

## 五、广告数据指标

| 指标 | 说明 | 健康值参考 |
|------|------|-----------|
| CTR（点击率）| 广告展示→点击的比率 | ≥0.5% |
| CVR（转化率）| 点击→购买的比率 | ≥10% |
| ACOS | 广告费用占销售额的比例 | 根据利润率而定，理想<15% |
| TACOS | 总销售额中广告贡献占比 | 越低越好 |
| RoAS | 广告支出回报率（销售额/广告费）| ≥3:1 |
| 花费（Spend）| 广告总消耗 | 监控不要超预算 |

### ACOS 计算公式
```
ACOS = (广告花费 / 广告带来的销售额) × 100%
```
**例子：** 广告花了$50，带来销售额$500 → ACOS = 10%

## 六、新品广告投放策略

### 第1阶段：自动广告（Day 1 ~ Day 14）
- 开启自动广告，让系统学习关键词和匹配人群
- 设置较低预算（$20~$50/天）
- 收集数据，不要手动干预

### 第2阶段：手动广告+精准关键词（Day 14 ~ Day 30）
- 分析自动广告数据，找出高表现关键词
- 开启手动广告，主打精准匹配
- 否定低效关键词（防止浪费预算）

### 第3阶段：持续优化
- 根据ACOS和转化数据调整竞价
- 开启SB/SD扩展流量来源
- 定期添加新的关键词

### 否定关键词
| 类型 | 说明 |
|------|------|
| 否定词组 | 包含该词组的搜索不展示 |
| 否定精确 | 完全匹配的搜索词不展示 |
- 使用否定词过滤不转化的搜索词，节省预算

## 七、ACOS优化技巧

1. **优化Listing**：广告带来流量，但如果Listing差则无法转化。先优化详情页再开广告。
2. **关键词筛选**：保留ACOS低的关键词，暂停/降低高出价关键词。
3. **竞价调整**：对于高绩效关键词提高竞价，低绩效的降低。
4. **匹配类型优化**：广泛匹配容易跑词，适时切换词组/精确匹配。
5. **时间节点**：根据广告报表，按周调整，不要每天频繁操作。
