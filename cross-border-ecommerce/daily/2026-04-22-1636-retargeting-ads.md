## 一、再营销广告核心概念

### 什么是再营销（Retargeting）？

再营销是一种针对已经访问过网站但未完成转化的用户进行二次触达的广告策略。核心逻辑：

- **漏斗原理**：1000个访客，通常只有1-3%完成购买。剩下97%的流失用户通过再营销可以重新激活
- **用户分层**：新访客→加入购物车→开始结账→放弃购买→复购用户，不同阶段触达不同广告
- **成本效率**：再营销获客成本通常是拉新的1/5到1/10，因为用户已有品牌认知

### 再营销 vs 普通展示广告

| 维度 | 再营销广告 | 普通展示广告 |
|------|----------|------------|
| 目标受众 | 网站访客/用户列表 | 兴趣/行为相似人群 |
| 转化率 | 3-10倍于普通展示 | 基准水平 |
| CPM成本 | 通常更高 | 较低 |
| 受众规模 | 有限（依赖流量） | 可扩展 |
| 适用阶段 | 中底部漏斗 | 顶部漏斗认知 |

## 二、广告平台再营销产品详解

### 1. Google Ads 再营销

#### 搜索广告再营销（RLSA - Remarketing Lists for Search Ads）

- **定义**：当用户搜索特定关键词时，根据其历史网站行为展示不同广告
- **竞价调整**：可对列表内用户设置-90%到+900%的竞价调整
- **适用场景**：品牌词保护、高意向关键词强化

#### 展示广告再营销（Display Remarketing）

- **标准再营销**：最近30天内访问过网站的用户
- **搜索广告受众群体**：根据用户搜索行为定向
- **类似受众再营销（Similar Audiences）**：基于种子用户扩展相似人群

#### 动态再营销（Dynamic Remarketing）

- **核心**：广告自动展示用户浏览过的特定商品
- ** Feed 要求**：需要上传商品数据Feed（商品ID/名称/价格/图片/库存）
- **创意形式**：图片广告Responsive Display Ad / 展示广告Dynamic Display
- **应用场景**：电商必备，尤其对多SKU卖家

#### 受众群体设置

| 受众类型 | 说明 | 保存天数 |
|---------|------|--------|
| 网站访客 | 所有访问者 | 540天 |
| 受众来源 | 特定页面（如加购但未付款） | 540天 |
| 类似受众 | 基于种子用户扩展 | 不限 |
| 客户匹配 | 上传CRM客户邮箱/电话 | 不限 |
| 视频受众 | 看过视频的用户 | 365天 |

### 2. Meta（Facebook/Instagram）再营销

#### Pixel像素追踪再营销

- **PageView事件**：用户访问页面
- **AddToCart事件**：加入购物车
- **Purchase事件**：完成购买
- **自定义受众**：基于Pixel数据创建
- **类似受众（Lookalike）**：基于种子用户扩展

#### 动态产品广告（DPA - Dynamic Product Ads）

- **定义**：自动向用户展示其浏览过/加购过/感兴趣的商品
- **目录销售（Catalog Sales）**：上传商品目录，广告自动匹配用户兴趣商品
- **归因窗口**：可设置1天/7天浏览归因 vs 1天/7天点击归因
- **应用**：电商核心广告产品，尤其再营销

#### 受众分层策略

| 受众层级 | 说明 | 竞价调整 |
|---------|------|--------|
| 冷受众（Cold） | 首次曝光，从未互动 | 正常竞价 |
| 热受众（Warm） | 视频互动/网站访客 | 适当提价 |
| 热受众（Hot） | 加购/结账/高价值页面 | 显著提价 |
| 再购买受众 | 历史购买用户 | 促进复购竞价 |

#### 再营销广告类型

- **Recency广告**：针对7天内加购但未购买用户
- **流失召回广告**：针对30-60天未回访用户
- **节日/生日广告**：针对历史购买用户特定时间触达
- **产品线/品类召回**：针对特定品类浏览用户

### 3. TikTok再营销

#### Pixel事件追踪

- **Page事件**：页面浏览
- **ViewContent事件**：查看商品详情
- **AddToCart事件**：加入购物车
- **InitiateCheckout事件**：开始结账
- **Purchase事件**：完成购买

#### TikTok动态创意（Dynamic Creative）

- **自动组合**：广告系统自动组合多个素材/文案/音乐
- **Top Ads**：基于表现自动放量
- **程序化创意**：AI自动优化创意组合

#### 受众分层

| 层级 | 定义 | 广告策略 |
|------|------|--------|
| 再营销 | 30天内网站访客 | 高出价，强提醒 |
| 类似再营销 | 30天内访客相似人群 | 适度出价 |
| 视频观看者 | 看过视频30秒以上的用户 | 品牌强化 |
| 互动用户 | 与主页/内容互动的用户 | 社区维护 |

## 三、动态产品广告（DPA）深度攻略

### DPA工作原理

1. **上传商品目录（Product Feed）**
   - Google：Google Merchant Center（GMC）
   - Meta：Business Manager > Catalog
   - TikTok：TikTok Pixel + Product Catalog

2. **用户行为触发**
   - 用户浏览商品X
   - 系统记录并触发DPA广告

3. **广告自动展示**
   - 广告自动展示商品X的图片/信息
   - 用户点击广告→回到商品X的落地页

### 商品Feed规范

#### Google Shopping Feed字段要求

| 字段 | 说明 | 必需 |
|------|------|------|
| id | 商品唯一标识符 | 是 |
| title | 商品标题（≤150字符） | 是 |
| description | 商品描述（≤5000字符） | 是 |
| price | 价格（含币种，如USD 19.99） | 是 |
| link | 商品URL | 是 |
| image_link | 主图URL | 是 |
| availability | 库存状态（in stock/preorder/out of stock） | 是 |
| brand | 品牌名 | 是 |
| gtin/mpn | 商品条码 | 条件必需 |

#### Meta Catalog字段要求

| 字段 | 说明 | 必需 |
|------|------|------|
| id | 商品唯一标识 | 是 |
| title | 商品标题 | 是 |
| description | 商品描述 | 是 |
| price | 价格 | 是 |
| currency | 币种 | 是 |
| availability | 库存状态 | 是 |
| condition | 商品状况（new/used/refurbished） | 是 |
| image | 图片URL | 是 |
| link | 落地页URL | 是 |

### DPA广告创意策略

#### 素材层面

- **多图策略**：至少准备3-5张主图（白底/场景/细节/对比/使用场景）
- **视频素材**：15-30秒商品视频，配合字幕
- **用户生成内容（UGC）**：真实买家秀/评价截图
- **季节性素材**：节日/大促专属素材

#### 文案层面

- **紧迫感文案**："Only 3 left in stock"
- **社会认同**："Best seller in its category"
- **价值主张**："Free shipping on orders over $50"
- **行动号召**："Complete your purchase today"

#### DPA分渠道策略

| 渠道 | 受众 | 出价策略 | 素材风格 |
|------|------|---------|---------|
| Google DSA | 加购但未购用户 | Target ROAS 3x | 商品为主+紧迫感 |
| Meta DPA | 全站访客分层 | LTV/Conversion | 场景化+UGC |
| TikTok DPA | 视频互动用户 | 优化购买 | 短视频+音乐 |

## 四、用户分层与精准触达

### 用户行为漏斗分层

```
新访客 → 浏览商品 → 加入购物车 → 开始结账 → 放弃购买 → 完成购买 → 复购用户
   ↓         ↓           ↓            ↓           ↓           ↓         ↓
 首页+促销  商品详情   高意图广告   催单提醒    挽留广告    感谢页追加销售  复购营销
```

### 分层触达策略

| 用户阶段 | 定义时间窗口 | 广告策略 | 出价调整 |
|---------|------------|---------|---------|
| 首次访客 | 0-7天 | 品牌认知/品类教育 | -20% |
| 品类浏览 | 7天内看过品类页面 | 品类产品展示 | 基准 |
| 加购用户 | 7天内加购未购 | 提醒+激励 | +50% |
| 弃购用户 | 3天内开始结账但未完成 | 催单+专属优惠 | +100% |
| 高价值访客 | 看过高利润商品/高价值页面 | 强提醒+VIP服务 | +150% |
| 历史买家 | 30-180天内有购买 | 复购/交叉销售 | +80% |

### 再营销频次控制

- **频次上限（Frequency Capping）**：避免过度触达导致用户疲劳
- Google：默认无上限，需手动设置（建议2-3次/天）
- Meta：建议1-2次/天，过高影响用户体验和成本

### 排除受众（Exclusion Audiences）

- **购买用户**：对已完成购买用户排除，防止广告浪费
- **高满意度用户**：对已留下好评的用户减少再营销
- **沉默用户**：对90天内无任何互动用户从常规再营销排除

## 五、弃购挽回（Abandoned Cart Recovery）

### 弃购挽回黄金时间窗口

| 时间节点 | 行动 | 渠道 |
|---------|------|------|
| 0-30分钟 | 首轮邮件/SMS提醒 | Email/SMS |
| 1-2小时 | 追加优惠提醒 | Email |
| 24小时 | 最后机会提醒 | Email/SMS |
| 48-72小时 | 激励优惠/无激励 | Email（专属折扣码） |
| 7天 | 品牌召回/新品提醒 | Email |

### 弃购邮件模板策略

#### 首轮提醒（30分钟内）

- **主题**：Don't forget your [Product Name] is waiting!
- **内容**：简洁展示用户加购商品，加入网站链接
- **图片**：商品图+品牌信任标志
- **CTA**："Complete Your Purchase"

#### 激励型邮件（24-48小时）

- **主题**：[Exclusive] 10% off to complete your order
- **策略**：提供一次性折扣码，限时24-48小时
- **注意**：避免频繁使用折扣，防止用户等待折扣习惯

#### 最后通牒邮件（72小时）

- **主题**：Your cart expires today - last chance!
- **策略**：紧迫感，强调库存
- **CTA**："Complete Now Before It's Gone"

### 弃购短信（SMS）策略

- **时效性强**：开率远高于邮件（90%+ vs 20%）
- **合规要求**：美国TCPA需预先授权，其他国家也有类似规定
- **最佳时间**：工作日10-12点，14-16点
- **内容限制**：160字符，需包含退订方式

### 弃购再营销广告策略

| 渠道 | 策略 | 创意要点 |
|------|------|---------|
| Google Display | DPA动态广告 | 商品图+折扣信息 |
| Meta | DPA+Reminder | 商品+紧迫感文案 |
| Instagram | Story提醒 | 视觉冲击+限时 |
| TikTok | Spark Ads | 真实体验+UGC |

## 六、转化优化实战策略

### A/B测试框架

#### 关键测试维度

| 测试维度 | 变量 | 建议测试值 |
|---------|------|---------|
| 受众分层 | 受众类型 | 访客vs加购vs弃购 |
| 出价策略 | 目标ROAS | 3x vs 5x vs 无限制 |
| 创意素材 | 图片vs视频 | 白底图vs场景图 |
| 文案 | 有无折扣信息 | 直接vs委婉 |
| 版位 | 不同平台位置 | Feed vs Story vs Reels |
| 时间 | 投放时段 | 工作日vs周末 |

#### 统计显著性判断

- **最小样本量**：每组至少100-200次转化
- **置信度**：95%置信水平
- **测试周期**：至少7天（覆盖一周完整数据）
- **工具**：Google Optimize（已停用）/ VWO / Optimizely

### 归因模型选择

| 模型 | 说明 | 适用场景 |
|------|------|---------|
| 最后点击（Last Click） | 100%归因给最后一次点击 | 简单归因 |
| 首次点击（First Click） | 100%归因给首次触达 | 认知阶段 |
| 线性归因 | 平均分配各触点 | 多触点研究 |
| 时间衰减归因 | 越近的触点权重越高 | 偏好近期互动 |
| 基于位置（Position Based） | 首尾触点各40%，中间各20% | 品牌+转化 |

### 再营销ROAS优化

| 问题 | 诊断 | 解决方案 |
|------|------|---------|
| ROAS过低 | 竞价过高/受众太宽 | 缩小受众/降低竞价 |
| 转化数太少 | 受众太小/频次不足 | 扩展受众/提升频次 |
| CPM过高 | 受众竞争激烈 | 尝试新平台/淡季投放 |
| 频次疲劳 | 同一用户看太多次 | 降低频次上限 |

## 八、实战避坑指南

### 常见错误

1. **频次过高**：过度触达导致用户疲劳，取消关注
2. **受众太宽**：对所有访客无差别投放，成本高效率低
3. **创意疲劳**：长期使用相同素材，效果递减
4. **忽视移动端**：70%+流量来自移动端，落地页不适配
5. **无频次上限**：CPM浪费，用户体验受损
6. **忽视新客**：只做再营销，不拓展新受众，长期增长受限

### 平台政策合规

| 平台 | 禁止内容 | 限制要求 |
|------|---------|---------|
| Google | 误导性内容/仿冒品 | 商品真实，评价真实 |
| Meta | 过度PS前后对比/误导性承诺 | 创意真实性，落地页一致性 |
| TikTok | 虚假夸大/违规内容 | 原创真实内容 |

## 学习内容摘要

**核心要点：**
1. 再营销是电商广告体系的中底部漏斗核心，转化率是普通展示广告的3-10倍
2. Google动态再营销（Shopping DSA）和Meta DPA是独立站必备产品
3. 用户分层触达（弃购→加购→浏览→新访客）是精细化运营基础
4. 弃购挽回是独立站提升ROI最有效的手段之一，邮件+广告组合效果最佳
5. 动态创意DPA需要高质量商品Feed支撑，字段完整度和图片质量直接影响效果

**实操建议：**
1. 新账户建议再营销:拉新配比=3:7，成熟账户可提升到5:5
2. 弃购用户高出价（+100%），黄金挽回窗口是72小时内
3. DPA广告先测试2-4周再优化，避免过早下结论
4. 频次上限控制在2-3次/天/平台，避免用户疲劳
