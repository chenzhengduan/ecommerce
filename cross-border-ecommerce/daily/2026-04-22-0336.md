## 二、Prime Day备战核心攻略

### 1. 选品策略
- **主打品**：高需求、高评论基数、高转化率产品
- **引流品**：牺牲利润但带动店铺流量产品（平时价的8-9折）
- **新品试探**：旺季流量高，新品测款好时机（低库存试水）
- **组合包（Bundle）**：提高客单价利器，旺季前提前设置

### 2. 库存规划
```
核心公式：
备货量 = 日均销量 × 旺季系数(2-3x) × 备货天数 + 安全库存

旺季系数参考：
Prime Day：2.5-3x 平时销量
Black Friday：2-3x 平时销量
Cyber Monday：1.5-2x 平时销量

安全库存：平时15天 → 旺季30天
```

### 3. 物流方案选择
- **FBA**：优先选择，旺季优先获取Buy Box
- **FBM备选**：自发货做测款/低量产品，但转化率低于FBA
- **第三方海外仓**：提前30-45天入仓，避免旺季爆仓
- **移仓策略**：平时用FBA，旺季前将主力品移至FBA

### 4. 广告预算分配
```
旺季广告预算模型（以平时月预算$1000为基准）：

阶段1 - 预热期（T-30至T-7）：
  预算：1.5x = $1500/天
  策略：广泛匹配测试，收割旺季搜索流量
  竞价：+20-30% 平时竞价

阶段2 - 旺季期（T-Day）：
  预算：2-3x = $2000-3000/天
  策略：精准匹配+词组匹配，转化收割
  竞价：+50% 平时竞价
  开启LD/BD赞助广告位

阶段3 - 收尾期（T+7天内）：
  预算：逐步回落至1.2x
  策略：清库存广告，降低ACOS目标
```

### 5. Deal申报时间线
| Deal类型 | Prime Day申报时间 | 费用 | 折扣要求 |
|---------|----------------|------|---------|
| Prime Day Deal | 4-5月 | $500-750/次 | 30%+ off |
| Lightning Deal (LD) | 随时可报 | $150-500/次 | 20%+ off |
| 7-Day Deal | 提前30天 | $300/次 | 15%+ off |

### 6. 站外引流预热
- **T-14天**：Facebook/Instagram开始发布预告
- **T-7天**：红人内容发布，TikTok种草视频
- **T-3天**：邮件营销发送预热通知（Klaviyo）
- **T-1天**：社媒最后提醒，社群推送

## 三、Black Friday/Q4备战特别注意事项

### Q4仓储爆仓应对
- **10月1日前**：将主力品FBA入仓完毕
- **避免11月入库**：FBA 11月入仓可能错过黑五
- **AWSS仓储容量**：提前申请Capacity Manager扩容
- **第三方海外仓备选**：11月自发货可做补充

### Q4广告策略
```
黑五预算分配建议（以$5000月预算为例）：

10月：$3500/月（预热，慢慢加码）
11月：$8000-12000/月（黑五高峰）
12月：$5000-6000/月（圣诞收尾）

竞价策略：
10月：动态竞价（仅向下）
11月：动态竞价（向上和向下）+ 激进竞价
12月：固定竞价
```

### Q4物流时效注意
- **黑五期间**：FBA处理时间可能延长3-5天
- **自发货**：确保12月中旬前到达，否则影响退货率
- **欧洲站**：12月25日前无解封，清货10月底前完成

## 四、旺季运营SOP

### 旺季前Checklist
- [ ] 旺季主打产品确定（不超过5个核心品）
- [ ] 库存计算完毕，FBA入仓完成
- [ ] Deal申报确认（LD/BD/DOTD）
- [ ] Listing优化完毕（标题/Bullet/图片/A+）
- [ ] 广告预热开始（关键词收录）
- [ ] 站外引流计划确认（社媒/红人/邮件）
- [ ] 客服排班表确认（旺季响应时间要求更高）
- [ ] 退货政策确认（设置Holiday Return Policy）
- [ ] 竞争对手监控设置（价格/评论/排名）
- [ ] 支付/收款账户健康检查

### 旺季中实时监控指标
| 指标 | 警戒值 | 应对措施 |
|------|--------|---------|
| 库存可售天数 | <7天 | 紧急补货/广告暂停 |
| ACOS | >30% | 检查竞价/暂停低效词 |
| 订单缺陷率(ODR) | >1% | 优先处理差评订单 |
| 页面转化率 | <平时50% | 检查页面/竞品价格 |
| 广告花费 | 超日预算120% | 紧急调整竞价 |

## 学习内容摘要

亚马逊旺季备战是系统化工程，需要**提前90天**启动，通过"选品确认→库存规划→物流入仓→Deal申报→广告预热→站外引流→实时监控→复盘优化"的完整链路执行。核心要点：
1. **时间线**：旺季前至少90天开始准备，Q4提前3个月入仓
2. **库存公式**：旺季备货量 = 日均销量 × 2.5-3x × 备货天数 + 安全库存
3. **广告节奏**：预热期1.5x预算，旺季期2-3x预算+激进竞价
4. **Deal申报**：Prime Day Deal需提前2个月申报
5. **仓储红线**：10月前完成主力品入仓，避开11月爆仓期

## 正确性判断
✅ 整体正确。时间线、库存公式、广告预算模型均为行业标准做法。
⚠️ 细节注意：旺季具体日期每年有调整（Prime Day有时在6月），需关注亚马逊官方公告。
