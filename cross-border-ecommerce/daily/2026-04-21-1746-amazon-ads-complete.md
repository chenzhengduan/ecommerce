## 一、亚马逊广告体系概述

### 1.1 亚马逊广告三大类型

亚马逊广告体系分为三大类型，分别承担不同的营销职能：

| 广告类型 | 全称 | 核心职能 | 适用场景 |
|---------|------|---------|---------|
| SP | Sponsored Products（商品推广） | 关键词搜索流量 | 新品推广、爆款打造 |
| SB | Sponsored Brands（品牌推广） | 品牌曝光+产品引流 | 品牌建设、购物意图流量 |
| SD | Sponsored Display（展示型推广） | 受众再营销+类目渗透 | 拉新、 Remarketing |

### 1.2 广告术语速查

| 术语 | 定义 | 计算公式 |
|------|------|---------|
| CPC | Cost Per Click（每次点击成本） | 花费 ÷ 点击数 |
| ACOS | Advertising Cost of Sales | 广告花费 ÷ 广告销售额 |
| TACOS | Total Advertising Cost of Sales | 广告花费 ÷ 总销售额 |
| RoAS | Return on Ad Spend | 广告销售额 ÷ 广告花费 |
| CTR | Click-Through Rate（点击率） | 点击数 ÷ 展示数 |
| CVR | Conversion Rate（转化率） | 订单数 ÷ 点击数 |
| 竞价 | Bid | 愿意为每次点击支付的最高价格 |

## 二、商品推广（SP广告）深度攻略

### 2.1 SP广告匹配类型

**广泛匹配（Broad Match）**
```
定义：用户搜索包含任意顺序的关键词（单复数、同义词、相关词）时展示
示例：关键词 "running shoes"
→ 触发：running shoes, shoes for running, jogging shoes, men running sneakers
优点：最大曝光，覆盖潜在客户
缺点：点击精准度低，可能带来无效花费
适用：新品测试、关键词挖掘阶段
```

**词组匹配（Phrase Match）**
```
定义：用户搜索包含完整关键词词组（可多出词但词序不能乱）时展示
示例：关键词 "running shoes"
→ 触发：red running shoes, running shoes for women, buy running shoes
→ 不触发：shoes running, running gear shoes
优点：中等精准度，曝光和精准平衡
缺点：覆盖范围比广泛匹配窄
适用：关键词验证后，精准扩展阶段
```

**精确匹配（Exact Match）**
```
定义：用户搜索与关键词完全一致（或极其接近的同义词）时展示
示例：关键词 "running shoes"
→ 触发：running shoes, running shoe
→ 不触发：red running shoes, running shoes for men
优点：最高精准度，点击质量高
缺点：覆盖范围最窄
适用：核心关键词运营、转化优化阶段
```

### 2.2 SP广告竞价策略

**动态竞价（Dynamic Bidding）**
- **只降低（Down Only）**：系统检测到转化可能性低时自动降低竞价，节省花费
- **只提高（Up and Down）**：系统检测到高转化可能性时提高竞价（最高+100%），抢量
- **固定竞价（Fixed Bids）**：不做任何动态调整，按设定竞价固定出价

**适用场景建议：**
```
新品冷启动期：固定竞价（先积累数据）
爆款打造期：动态-只提高（抢流量）
稳定期：动态-只降低（优化ACOS）
清仓/甩货：固定竞价（控制成本）
```

**竞价调整策略**
```
基于表现调整竞价：
├─ ACOS < 目标值 → 可适当提高竞价（+10-20%）
├─ ACOS > 目标值 → 降低竞价（-10-20%）或暂停
├─ 有曝光无点击 → 检查主图/价格/评分，优化Listing
└─ 有点击无转化 → 检查Listing详情页、活动竞争力
```

### 2.3 SP广告结构设计

**广告组结构原则：**
```
一个广告活动（Campaign）
    ↓
多个广告组（Ad Group）— 按产品/品类分组
    ↓
多个广告组内：自动广告 + 手动广告

推荐结构示例：
Campaign: [品类名] - 自动
    └ Ad Group: [ASIN A] - 自动

Campaign: [品类名] - 手动 - 核心词
    └ Ad Group: [ASIN A] - 手动广泛
    └ Ad Group: [ASIN A] - 手动词组
    └ Ad Group: [ASIN A] - 手动精确
```

**否定关键词管理**
```
否定广泛：否定整体大词（如" shoes"）
否定精确：否定具体低效词（如"shoes for women"）

每周检查否定关键词列表：
├─ 有花费但无转化的词 → 否定
├─ 花费高但ACOS超标的词 → 降低竞价或否定
└─ 与产品不相关的词 → 否定
```

## 三、品牌推广（SB广告）深度攻略

### 3.1 SB广告格式类型

| 格式 | 描述 | 要求 | 适用场景 |
|------|------|------|---------|
| 商品集 | 展示3个及以上产品+自定义标题 | 至少3个ASIN | 产品线推广 |
| 品牌旗舰店焦点 | 流量导向品牌旗舰店 | 已完成品牌备案 | 品牌认知推广 |
| 视频 | 单产品视频广告 | 单个ASIN | 视觉冲击强产品 |

### 3.2 SB广告投放策略

**关键词投放：**
- 竞品品牌词（"相比[竞品品牌]，我们的..."）
- 行业词/品类词（"户外运动鞋"类目词）
- 自家品牌词（保护品牌流量，被竞品截流时）

**商品投放：**
- 投放竞品ASIN（竞争相似受众）
- 投放自家ASIN（关联销售）

### 3.3 SB vs SP的核心区别

| 维度 | SB品牌推广 | SP商品推广 |
|------|-----------|-----------|
| 目标 | 品牌曝光 + 转化 | 精准转化 |
| 展示位置 | 搜索结果顶部/竞品旁边 | 搜索结果内部/产品详情页 |
| 可引导至 | 品牌旗舰店 | 产品详情页 |
| 最低预算 | $100/天 | 无限制 |
| 品牌要求 | 必须完成品牌备案 | 不需要 |

## 四、展示型推广（SD广告）深度攻略

### 4.1 SD广告定向类型

**受众定向：**
- **再营销定向（Remarketing）**：访问过产品页面但未购买的用户
- **相似受众（Similar Audiences）**：购买过同类产品的用户
- **生活方式定向（Lifestyle）**：按兴趣/行为分类的人群
- ** проживання/人口统计学定向**：按年龄/性别/收入定向

**商品投放：**
- 投放竞品ASIN（触达竞品受众）
- 投放类目/品类（触达品类兴趣用户）

### 4.2 SD广告适用场景

```
新品拉新：触达品类兴趣用户，不依赖关键词
再营销：触达看过产品但未购买的用户，提高转化
竞品截流：投放竞品ASIN，抢竞品受众
清仓甩货：触达广泛受众，促进购买决策
```

### 4.3 SD广告竞价策略

- 访问量定价（CPM-based）：按千次展示付费，适合品牌曝光
- 转化量定价（eCPC）：按点击付费，效果好时自动提价，适合转化

## 五、广告数据归因与衡量

### 5.1 归因窗口类型

| 窗口 | 说明 | 适用场景 |
|------|------|---------|
| 7天归因 | 点击后7天内购买算作广告贡献 | 标准衡量 |
| 14天归因 | 点击后14天内购买算作广告贡献 | 高客单价产品 |
| 30天归因 | 点击后30天内购买算作广告贡献 | 决策周期长的产品 |

### 5.2 核心衡量指标体系

| 指标 | 计算 | 健康标准 |
|------|------|---------|
| CPC | 花费 ÷ 点击 | < $1（竞争激烈类目<$2）|
| CTR | 点击 ÷ 展示 | > 0.5%（具体看类目）|
| CVR | 订单 ÷ 点击 | > 10%（具体看类目）|
| ACOS | 花费 ÷ 销售额 | < 30%（竞争激烈<40%）|
| TACOS | 花费 ÷ 总销售额 | < 15% |
| RoAS | 销售额 ÷ 花费 | > 3x |

### 5.3 广告组合拳打法

```
新品期（0-30天）：
├─ SP自动广告（积累数据）
├─ SP手动广泛（测试关键词）
└─ 预算：$30-50/天

成长期（30-90天）：
├─ SP手动精确（跑核心词）
├─ SB品牌推广（提升品牌）
└─ 预算：$50-100/天

稳定期（90天+）：
├─ SP手动（持续优化ACOS）
├─ SB品牌（品牌维护）
├─ SD再营销（提升复购）
└─ 预算：$100+/天，按ROI分配
```

## 六、广告优化实战技巧

### 6.1 主图优化

广告点击率（CTR）的核心因素是主图：

```
高CTR主图特征：
├─ 产品突出（不杂乱）
├─ 使用场景（让用户代入）
├─ 差异化（颜色/角度/概念）
└─ 简洁清晰（快速传达卖点）

测试方法：
A/B测试不同主图（每个运行14天+）
保留CTR最高的，继续测试下一个变量
```

### 6.2 广告时段优化

根据数据调整广告投放时段：
```
分析时段报表（按小时）：
├─ 高转化时段 → 提高竞价
├─ 低转化时段 → 降低竞价或暂停
└─ 无数据时段 → 持续观察

注意：只参考有足够数据量的时段，避免样本不足误判
```

### 6.3 关键词质量判断

| 信号 | 判断 | 行动 |
|------|------|------|
| 高曝光、高点击、高转化 | 核心词，加价 | +20%竞价 |
| 高曝光、高点击、低转化 |Listing问题 | 优化详情页 |
| 低曝光、高点击、高转化 | 精准词，加价 | +20%竞价 |
| 低曝光、低点击、低转化 | 词不对/预算不够 | 换词或提预算 |
