## 一、站内广告（Amazon PPC）

### 广告类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **Sponsored Products** | 产品搜索结果广告，按点击付费 | 新品推广、爆款助推 |
| **Sponsored Brands** | 品牌展示广告，含品牌LOGO+3个产品 | 品牌曝光、品类竞争 |
| **Sponsored Display** | 再营销广告，站内外投放 | 复购、潜在客户拉新 |

### 广告结构策略

**冷启动期（0-3个月）**
- 手动精准匹配：选择高相关度关键词，bid 略高于建议价
- 同 ASIN 竞争：对标竞品 listing 做防御性投放
- 预算控制：日预算 50-100 美元，测试 2 周

**成长期（3-6个月）**
- 自动广告+手动广泛：跑词，扩大流量池
- 否定无效词：持续优化关键词列表
- 提高预算：将表现好的广告组预算翻倍

**稳定期（6个月+）**
- 结构化广告组合：每个产品 3-5 个广告组，区分匹配类型
- Bidding Strategy：用固定竞价或 Up & Down，配合acos目标
- 品牌广告：加入品牌词保护

### ACOS 优化核心

**ACOS = 广告花费 / 广告销售额 × 100%**

| ACOS 区间 | 评价 | 优化方向 |
|-----------|------|----------|
| < 15% | 优秀 | 可以适当加预算 |
| 15-30% | 健康 | 维持，持续优化 |
| 30-50% | 偏高 | 优化关键词、落地页 |
| > 50% | 危险 | 检查产品是否适合做广告 |

## 二、站外引流（独立站 + 社媒）

### 主要流量来源

| 渠道 | 特点 | 适合品类 |
|------|------|----------|
| **Facebook/Instagram** | 社交广告，精准定向 | 快消、时尚、生活方式 |
| **Google Ads** | 搜索广告，意图明确 | 标准品、刚需品 |
| **TikTok** | 短视频，内容营销 | 年轻用户、新奇特 |
| **Pinterest** | 图片发现电商 | 家居、婚礼、DIY |
| **YouTube** | 视频测评、种草 | 3C、工具、户外 |

### 独立站 + Facebook 投放基础

**投放漏斗**
```
 awareness（品牌认知）→ consideration（产品考虑）→ conversion（转化）
```

**受众定向**
- **Custom Audience**：上传老客邮箱，再营销
- **Lookalike**：基于老客拓展相似人群（1-3% 最接近）
- **兴趣定向**：选择相关品类兴趣标签

**创意要点**
- 前3秒抓眼球：开场要有钩子
- 字幕+无声友好：大多数用户在静音观看
- CTA 明确："Shop Now" / "Learn More"

## 三、搜索引擎优化（SEO）

### Amazon SEO

**排名因子（重要性排序）**
1. **相关性**（标题、要点、搜索词）—— 关键词匹配度
2. **转化率**（销量、转化率）—— 亚马逊最看重的信号
3. **账号权重**（评分、ODR、订单缺陷率）—— 健康度
4. **物流方式**（FBA > FBM）—— 物流权重
5. **丰富度**（A+、视频、QA）—— 内容丰富度加分

**关键词布局**
```
标题：主关键词 + 品牌 + 型号 + 核心卖点（60字符内）
要点：5个要点，前3个包含核心关键词
Backend Keywords：250字节，不重复标题中的词
```

### Google SEO（独立站）

**技术SEO**
- 网站速度：< 3秒加载，核心指标（LCP < 2.5s）
- 移动优先：Google 优先索引移动版
- HTTPS / SSL 证书
- 结构化数据（Schema Markup）

**内容SEO**
- 原创内容：Google 偏好独特、有价值的内容
- 关键词布局：标题、H1、H2、正文自然分布
- 内链结构：相关文章互相链接
- 外链建设：高权重网站的外链引用

## 四、测评与评价管理

### 合法测评渠道

| 渠道 | 说明 |
|------|------|
| **Vine Voice** | 亚马逊官方测评，付费（200美元/产品），可信度最高 |
| **Early Reviewer Program** | 亚马逊官方，小额奖励换取评价 |
| **真实购买者** | 站外社媒粉丝、KOL、忠实客户 |

### 评价管理策略

**获取评价**
- 产品卡中附感谢卡（不诱导，提供指引）
- 邮件跟进：发货后 5-7 天发送请求邮件
- 简化评价流程：提供截图+链接

**负面评价处理**
- 第一时间回复：礼貌、专业
- 申请编辑：如果是瑕疵问题，可申请顾客编辑评价
- 利用负面评价优化产品：差评是产品改进的输入

## 五、红人营销（KOL/KOC）

### 平台选择

| 平台 | 适合合作类型 |
|------|--------------|
| YouTube | 深度测评、开箱、教程（长视频） |
| Instagram | 图片种草、Stories 带货 |
| TikTok | 短视频带货、挑战赛 |
| 独立博客 | 搜索引擎长尾流量 |

### 合作模式

- **纯佣**：按实际销售额提成（5-20%），低风险
- **付费+佣金**：基础费用 + 销售提成
- **付费一口价**：固定费用，适合腰部以上 KOL
- **赠品合作**：免费寄样，小 KOC 或微网红

### 合作避坑

- 查真伪：粉丝数、互动率（一般 > 3% 为正常）
- 签合同：明确发布内容要求、时间节点、版权归属
- 跟踪链接：所有合作带 UTM 追踪，量化 ROI

## 六、邮件营销（EDM）

### 邮件序列

**新订阅用户**
```
Day 0：欢迎邮件（介绍品牌+首单优惠）
Day 2：产品推荐（热卖品）
Day 5：社媒互动（关注我们）
```

### 关键指标

| 指标 | 健康标准 |
|------|----------|
| 发信率（Open Rate） | > 20% |
| 点击率（Click Rate） | > 5% |
| 转化率（Conversion Rate） | > 2% |
| 退订率（Unsubscribe Rate） | < 0.5% |

## 学习内容摘要

营销推广是跨境电商流量获取的核心，分站内（Amazon PPC）和站外（社媒、SEO、红人）两大体系。站内广告要关注 ACOS 健康值，站外引流核心是内容+精准定向。SEO 是长期投资，需要持续优化。测评、红人、邮件营销是辅助工具，配合促销节点形成合力。旺季前要提前 2-3 个月规划备货和广告预算。

**较好理解**：实操性很强，光看理论不够，需要实际投放才能真正掌握。建议从 Amazon PPC 手动精准广告开始练手，先小预算测试，再逐步放大。

## 我的判断

营销推广是跨境电商的"发动机"，但也是烧钱最快的环节。核心原则：
1. **站内广告优先**：Amazon PPC 是跨境卖家最该精通的技能
2. **先优化再放量**：ACOS 不健康时不要盲目加预算
3. **站外流量是复利**：SEO 和社媒积累是长期资产
4. **旺季提前规划**：广告预算、促销工具都要在旺季前到位
5. **数据驱动决策**：所有广告决策基于数据，而非感觉
