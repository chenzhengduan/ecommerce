## 二、Inventory Performance Dashboard（库存绩效仪表盘）

### 2.1 IPI四维评分体系

亚马逊库存绩效指数（IPI）由四个维度构成：

1. **过量库存因子**（Excess Inventory Factor）
   - 评估标准：超90天可售的库存占比
   - 优化方式：进行FBA Liquidations、清仓促销、移仓移除
   - 健康值：库存周转天数 60~90天

2. **现货率因子**（In-Stock Factor）
   - 评估标准：当前有货的ASIN占比
   - 优化方式：保持安全库存公式计算补货点
   - 健康值：现货率 ≥ 95%

3. **待处理库存因子**（Unprocessed Inventory Factor）
   - 评估标准：入库但尚未上架的库存
   - 优化方式：确保FBA发货流程顺畅

4. **FBA库存率因子**（FBA Inventory Rate Factor）
   - 评估标准：FBA库存占总库存的比例
   - 优化方式：逐步将自发货转为FBA

### 2.2 安全库存公式

```
补货点 = (日均销量 × 备货天数) + 安全库存
安全库存 = (最大日均销量 - 日均销量) × 备货天数 × 备货因子(通常1.5~2)
```

### 2.3 库龄分级（Age Grade）

| 库龄 | 状态 | 建议处理方式 |
|------|------|-------------|
| 0~90天 | 健康 | 正常销售 |
| 91~180天 | 警告 | 开始促销或移仓 |
| 181~270天 | 危险 | 清仓Liquidations |
| 271~365天 | 超龄 | 移除或大幅折扣 |
| >365天 | 超期 | 强制移除（亚马逊会收高额超期费）|

## 三、广告绩效仪表盘

### 3.1 核心指标体系

| 指标 | 定义 | 健康标准 |
|------|------|---------|
| ACOS | 广告支出÷广告销售收入 | < 30%（竞争激烈类目可接受30~40%）|
| TACOS | 广告支出÷总销售收入（含自然销售）| < 15% |
| RoAS | 广告销售收入÷广告支出 | > 3.0 |
| CTR | 点击率 | > 0.5%（类目不同差异大）|
| CVR | 转化率 | > 10%（取决于类目）|

### 3.2 ACOS问题排查思路

- **ACOS偏高但销量稳定**：检查竞价是否过高，尝试降低竞价或调整定向
- **ACOS偏高且销量下降**：可能是关键词不够精准，需要优化匹配方式
- **ACOS正常但TACOS偏高**：说明自然排名靠前，广告依赖度低 → 健康状态
- **新ASIN ACOS爆高**：新品期（1~4周）通常ACOS高，属于正常现象，主要看趋势是否逐步下降

### 3.3 广告结构优化

- **自动广告**：跑词+识别长尾词，积累数据后迁移到手动广告
- **手动广告（精准/词组/广泛）**：精准词做高竞价+精准匹配，长尾词做低竞价+广泛匹配
- **SB品牌广告**：新品期主推品牌词，保护品牌流量
- **SD展示型广告**：再营销召回人群，站外引流

### 3.4 工具辅助

- **Perpetua**：AI广告自动化投放，支持全自动竞价优化
- **Sellics**：整合广告+财务数据，支持ACOS趋势追踪
- **DataHawk**：亚马逊官方SP API数据，广告+库存+财务一体化
- **Teikametrics**：支持全层级广告组合分析

## 四、利润分析仪表盘

### 4.1 利润结构拆解

```
销售收入
- 平台佣金（约15%）
- FBA配送费
- 头程运输费
- 商品成本
= 毛利（约30%+为健康）
- 广告费用
- 仓储费用
- 退货损耗
- 促销折扣
= 净利
```

### 4.2 关键利润率基准

| 指标 | 健康标准 | 警告线 |
|------|---------|--------|
| 毛利率 | ≥ 30% | < 20% |
| 广告占营收比 | 10~20% | > 25% |
| 仓储费占营收比 | 3~5% | > 8% |
| 净利率 | ≥ 10% | < 5% |

### 4.3 FBA费用构成（2026年参考）

| 费用项 | 计费方式 |
|--------|---------|
| 月度仓储费 | 每立方英尺（标准件/超大件）|
| 超期仓储附加费 | 库龄>271天起收（比普通仓储费高数倍）|
| 配送费 | 按重量和体积计算 |
| 移除/销毁费 | 按件收取 |
| FBA Liquidations | 回收价约为处理费的60~70% |
| 退货处理费 | 接受退货时收取 |

### 4.4 利润分析工具

- **DataHawk**：直接接入SP API，展示利润仪表盘，支持按ASIN/店铺/时间维度查看
- **Sellics**：财务+广告整合分析，支持竞争对手利润对标
- **Helium10 Cerebro**：结合关键词反查和利润估算
- **自建BI（Looker Studio + SP API）**：适合中大型卖家，自定义程度高

## 五、三仪表盘联动逻辑

```
库存管理（补货）
    ↓
销量预测 → 广告绩效监控（ACOS/RoAS）
    ↓
利润分析（毛利率/净利率）
    ↓
选品调整（高毛利产品加码，低毛利产品减投）
```

## 六、学习摘要

**核心要点：**
- 精细化运营三大核心：库存绩效（不要断货/不要积压）、广告绩效（ACOS合理控制）、利润分析（净利率为王）
- IPI四维评分体系是库存管理基础，安全库存公式是日常运营必备
- 广告分析不能只看ACOS，必须结合TACOS和自然销售占比判断真实健康度
- 利润分析需拆解到每ASIN级别，才能发现真正的"亏损黑洞"
- 工具链：Perpetua/Sellics/DataHawk覆盖广告+财务，RestockPro/Stocky覆盖库存
