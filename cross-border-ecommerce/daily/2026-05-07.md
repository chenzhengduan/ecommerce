# 493-跨境电商营收计算与利润分析

## 学习日期
2026-05-07

## 主题
跨境电商营收计算与利润分析

## 学习内容摘要

### 1. 营收公式
**销售收入 = 销售价格 × 销售数量**

对于亚马逊等电商平台：
**净营收 = (商品售价 - 平台佣金 - 平台配送费) × 销售数量**

### 2. FBA利润计算公式
```
单品利润 = 售价 - 采购成本 - 头程运费 - FBA仓储费 - 平台佣金(通常15%) - 配送费 - 运营成本
```

### 3. 关键指标
- **毛利率** = (营收 - 成本) / 营收 × 100%
- **净利润率** = 净利润 / 营收 × 100%
- **ROI** = (收入 - 投资成本) / 投资成本 × 100%
- **周转率** = 销售额 / 库存成本（越高越好）

### 4. 成本构成
1. **产品成本**：采购价 + 包装
2. **物流成本**：头程(中国→海外仓) + 尾程(FBA配送)
3. **平台成本**：佣金 + 配送费 + 仓储费
4. **运营成本**：广告 + VAT(欧洲) + 退款处理 + 损耗

### 5. 定价策略
- **成本加成法**：成本 × (1 + 期望利润率)
- **市场对标法**：参考竞品价格，保证有竞争力
- **动态调价**：根据库存、季节、竞品调整

## 判断（对不对？）

✅ 基本公式正确
✅ 亚马逊佣金通常为15%(具体类别略有差异)
⚠️ FBA仓储费计算较复杂，按体积和存放时长计费
