## Why

当前知识库重点围绕跨境电商和 Amazon 体系展开，还没有独立承载阿里巴巴电商知识的目录和网页入口。新增阿里巴巴电商知识库可以系统沉淀 1688、Alibaba.com、淘系/天猫、阿里国际站、供应链、询盘、批发、履约、支付和平台规则等知识，并通过现有 MkDocs 网页统一浏览。

## What Changes

- 新增一个独立的阿里巴巴电商 Markdown 知识库目录，和现有 `cross-border-ecommerce/` 并列或在现有 docs 结构中清晰分区。
- 为阿里巴巴电商知识库建立首页、主题地图、基础模块和后续刷新机制。
- 将阿里巴巴电商知识库接入现有 MkDocs Material 网页，让它能通过本地网页浏览、搜索和左侧目录访问。
- 建立初始知识体系，覆盖平台基础、1688、Alibaba.com、阿里国际站运营、供应链采购、商品发布、询盘转化、交易履约、支付结算、营销获客、数据分析、风控合规等主题。
- 不破坏现有跨境电商知识库和网页访问方式。

## Capabilities

### New Capabilities

- `alibaba-ecommerce-knowledge-base`: 定义阿里巴巴电商知识库的目录结构、初始内容范围、MkDocs 网页接入、主题地图和长期维护方式。

### Modified Capabilities

- 无。

## Impact

- 影响内容：新增阿里巴巴电商知识库目录、首页、主题地图、初始 Markdown 笔记和刷新/维护文档。
- 影响配置：需要调整 MkDocs 配置，使本地网页能够同时浏览跨境电商和阿里巴巴电商知识库。
- 影响工具：现有知识刷新工具可能需要泛化，支持不同知识库目录的扫描和报告。
- 依赖：涉及平台规则、费用、交易保障、物流、支付和合规等时效性内容时，实施阶段需要查验阿里巴巴、1688、Alibaba.com 等官方或高质量来源。
